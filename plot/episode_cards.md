## EP-001: ログイン ─ 新世界の扉

| 項目 | 内容 |
|------|----|
| タイトル | 「ログイン ─ 新世界の扉」 |
| VR内日数 | Day 0 |
| 視点 | Ren（一人称） |
| 推定文字数 | 約3.5万字 |
| **トーン** | **★★★★★ 光（純粋な喜び。不安要素ゼロ）** |

| 要素 | 詳細 |
|------|----|
| **目的** | 世界観・主人公・HVD体験の紹介。読者を「この世界に入りたい」と思わせる |
| **障害** | 些細なバグ（テスターとしての日常）。デバッグエリアのモンスターとの遭遇 |
| **転換** | なし。純粋に「良い一日だった」で終わる |
| **情報開示** | HVD五感体験、Arcadia7リージョン概要、RSS、テスターコミュニティ |
| **フック** | Renの独白：「明日も楽しみだ」（**不安は一切ない**） |

### シーン構成

1. **オープニング**: 現実世界。Ren がNH社テスト施設でカプセルに入る準備。技術的な説明を自然に織り込む
2. **ログイン**: 五感が切り替わる体験の描写（黒→光→色彩→音→風→匂い）
3. **Central Nexus探索**: 浮遊都市の壮大さ。NPCとの対話。RSSの初体験
4. **デバッグ任務**: The Undercroft入口付近でバグ報告任務。戦闘システム紹介
5. **他テスターとの出会い**: Pixel（明るい少年）、Mila（穏やかな研究者）との交流。**Advocateが「良いチームだな」と温かく声をかける**
6. **夕暮れ**: Arcadiaの夕日を見ながら「いい世界じゃないか」と独白。**Pixelが「ねぇRen、俺、最近ログインする前のこと、あんまり思い出せないんだよね」と何気なく言う**。Renは「没入しすぎだろ」と笑う
7. **エンディング**: 就寝（VR内睡眠）。「明日も楽しみだ」

| **転換** | 世界が消えていく中、各キャラクターが最後の言葉を交わす → 暗転 → 覚醒 |
| **情報開示** | 停止シーケンスの全プロセス、副作用、後日談 |
| **フック（エピローグ末尾）** | Ren がリハビリ中にスマホを見る。Lynx からのメッセージ：「ねぇ、現実の空って、こんなに青かったっけ？」 |

### エピローグ構成

1. カプセル覚醒。光、液体、呼吸の感覚
2. 病院。身体のリハビリ（筋力低下）
3. NH社の記者会見（報道映像）
4. テスター同窓会（数ヶ月後。RSS `#memorial` が現実のWebサイトとして再現されている）
5. Ren とLynx が現実で初めて顔を合わせる。「……Lynxか？」「Renね。思ったより普通じゃん」
6. 最終行：Ren が夜空を見上げる。星が、Arcadiaの星と同じに見える。
