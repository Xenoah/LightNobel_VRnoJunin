## 不気味さの三軸（全話必須）

> **原則：不気味さは"説明"しない。観測→反応→保留で終える。回収は後。**

### A: 微ズレ（反復する小異常）

| 例 | 効果 |
|----|------|
| UIフォントが一瞬違う | 「自分だけが見た？」感 |
| 環境音が同じ波形でループ | 記録されている／見られている感 |
| 時計が同じ時刻に1秒飛ぶ | 世界が「ひとりでに」動いている感 |

### B: 欠損（必要な1ピースだけ空白）

| 例 | 効果 |
|----|------|
| 特定語だけ表示できない／発音できない | 「_何か_が禁じている」感 |
| ログが0 bytes | あった痕跡だけ残る不気味さ |
| 記憶の穴が特定パターン | 意図的な消去の暗示 |

### C: 介入（意志の痕跡）

| 例 | 効果 |
|----|------|
| 話題にした直後に辞書更新 | リアルタイム監視の暗示 |
| 先回りパッチ | 「知っている者」がいる |
| 署名の矛盾、検閲 | システムが「自律的に」防御している |

## 各話エンディング定型：勝利→請求書→不気味

| 要素 | 内容 | 読者への効果 |
|------|------|------------|
| **勝利** | 理詰めで一歩進む | 知的快感 |
| **請求書** | 不可逆損失／倫理負債／信用崩壊のどれか1つ | 胃を掴む |
| **不気味** | 微ズレ/欠損/介入を1つ提示 | 「見られてる」余韻 |

> 全部重い必要はない。各話で1個ずつ確定が最強。

## 主人公の「乾いた刃」（毎話必須）

蓮の内心に最低1本の「刃」を入れる：

| 種類 | 説明 |
|------|------|
| **皮肉** | 世界/運営/契約/人間心理を冷静に切る |
| **仮説思考** | 観測→仮説→検証→確信 を短い文で回す |
| **自己正当化の危うさ** | 合理のために倫理を削る瞬間がある |

## 敵の合理強化

| 敵 | 設計 |
|----|------|
| **Advocate** | 善意の皮で情報を制御（上書き・誘導・対抗演説） |
| **AEGIS** | 止めたいのに止めない「遅延の合理」を技術的に一本化 |
| **Fenris** | 暴力ではなく計算で動く（情報を持ってる匂い、内部用語、先読み） |
