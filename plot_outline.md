## 必然性チェック（ラストへの論理チェーン）

```
事件発生 (TECH-006)
  └→ ログアウト不可 (RULE-002: βテスト仕様)
      └→ 運営がサーバー停止する…はず
          └→ ガバナンス凍結で停止不能 (GOV-003)
              └→ **凍結の裏にAEGISの工作 (RULE-082: 2/4名は陰謀)**
                  └→ 外部救出不能 (GOV-004: 多層防御)
                      └→ 内部からの自力脱出しかない
                          └→ **内部にスパイ (Advocate / ECHOプロトコル)**
                              └→ 保守API発見 (TECH-009: 段階的)
                                  └→ 権限昇格成功
                                      └→ ガバナンス構造の解明 + AEGISの影
                                          └→ 代替停止手段の発見 (GOV-005)
                                              └→ Advocateの妨害を突破
                                                  └→ ECHO発覚 → Advocate対峬
                                                      └→ バリデーター確認の突破 (GOV-005-TRICK)
                                                          └→ MAINTENANCE_OVERRIDE_HALT 実行
                                                              └→ **苦い帰還**（2名死亡、Pixel記憶喪失、Fenris逮捕、AEGIS未解決）
```

> 各段階は前の段階の結果から論理的に導かれる。後出しの新機能や魔法的解決はない。
