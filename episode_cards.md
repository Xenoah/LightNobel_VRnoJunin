## EP-001: ログイン ─ 新世界の扉

| 項目 | 内容 |
|------|----|
| タイトル | 「ログイン ─ 新世界の扉」 |
| VR内日数 | Day 0 |
| 視点 | Ren（一人称） |
| 推定文字数 | 約3.5万字 |

| 要素 | 詳細 |
|------|----|
| **目的** | 世界観・主人公・HVD体験の紹介。読者を「この世界に入りたい」と思わせる |
| **障害** | 些細なバグ（テスターとしての日常）。デバッグエリアのモンスターとの遭遇 |
| **転換** | NCBログに「見覚えのない書き込みパターン」を発見するが、仕様範囲と判断 |
| **情報開示** | HVD五感体験、Arcadia7リージョン概要、BSS、テスターコミュニティ |
| **フック** | 「ログアウトボタン？ あぁ、βだからまだ実装されてないんだ。まぁ72時間テストだし、終わったら運営が落とすだろ」 |

### シーン構成

1. **オープニング**: 現実世界。Ren がNH社テスト施設でカプセルに入る準備。技術的な説明を自然に織り込む
2. **ログイン**: 五感が切り替わる体験の描写（黒→光→色彩→音→風→匂い）
3. **Central Nexus探索**: 浮遊都市の壮大さ。NPCとの対話。BSSの初体験
4. **デバッグ任務**: The Undercroft入口付近でバグ報告任務。戦闘システム紹介
5. **他テスターとの出会い**: Pixel（明るい少年）、Mila（穏やかな研究者）との交流
6. **夕暮れ**: Arcadiaの夕日を見ながら「いい世界じゃないか」と独白。NCBログの異常を確認するが放置
7. **エンディング**: 就寝（VR内睡眠）。「明日も楽しみだ」

| 項目 | 内容 |
|------|----|
| タイトル | 「静かな異変」 |
| VR内日数 | Day 1-2 |
| 視点 | Ren（一人称）+ 現実世界カットイン（三人称） |
| 推定文字数 | 約3.5万字 |

| 要素 | 詳細 |
|------|----|
| **目的** | 不安の醸成。「何かがおかしい」を読者に感じさせる |
| **障害** | ログアウトボタンが見つからない。運営への問い合わせが無応答 |
| **転換** | BSS `#news` に初のニュースが流入。「VR関連企業に重大事故か」 |
| **情報開示** | BSSチャンネル構成、NFC仕組み、テスター間の温度差 |
| **フック** | ニュースの見出し：「Nexus Horizons社、βテストで重大インシデント。詳細は明らかにされず」 |

### シーン構成

1. **朝**: Day 1。通常のテスト作業。だが運営NPCが応答しない
2. **ログアウト探し**: メニューを隅々まで探すが見つからない。他テスターも同様
3. **BSS上の不安**: `#general` に「ログアウトできない人いる？」の投稿が増加
4. **【現実カットイン】**: NH社モニタリングルーム。エレナ・ヴァイスが異常を検知。クロスがパニック
5. **Day 2朝**: BSS `#news` にニュースが流入。まだ断片的
6. **Mila との会話**: NPC対話のパターンが変。「このNPC、スクリプトじゃない応答をしてる」
7. **不安の夜**: テスターたちが群れ始める。Ren は冷静を装うが内心は動揺

| 視点 | Ren（一人称）→ エピローグで複数視点 |
| 推定文字数 | 約4万字 |

| 要素 | 詳細 |
|------|----|
| **目的** | カタルシス。余韻のある結末 |
| **障害** | 停止シーケンスの不確実性（本当に安全に戻れるのか？） |
| **転換** | 世界が消えていく中、各キャラクターが最後の言葉を交わす → 暗転 → 覚醒 |
| **情報開示** | 停止シーケンスの全プロセス、副作用、後日談 |

### エピローグ構成

1. カプセル覚醒。光、液体、呼吸の感覚
2. 病院。身体のリハビリ（筋力低下）
3. NH社の記者会見（報道映像）
4. テスター同窓会（数ヶ月後。BSS `#memorial` が現実のWebサイトとして再現されている）
5. Ren とLynx が現実で初めて顔を合わせる。「……Lynxか？」「Renね。思ったより普通じゃん」
6. 最終行：Ren が夜空を見上げる。星が、Arcadiaの星と同じに見える。
