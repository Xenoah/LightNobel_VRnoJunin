## 派閥詳細

### FACT-001: Exodus（帰還派）

| 項目 | 詳細 |
|------|----|
| 結成時期 | Day 5（派閥としての正式結成。それ以前は個人の動き） |
| 理念 | 技術的手段で内部から脱出する。暴力は最終手段 |
| 組織構造 | 当初はRen中心 → Day 15以降は合議制（技術班/情報班/防衛班の3部門） |
| 技術班 | 保守API解析、権限昇格、ブロックチェーン分析（約8名のエンジニア） |
| 情報班 | RSS分析、NCE活用、外部ニュースの解釈（約4名） |
| 防衛班 | Berserkers からの攻撃への防衛、拠点警備（約8名） |
| 課題 | 技術者の絶対数不足。非技術者の信頼獲得。進捗が見えにくいことへの不満 |

### FACT-002: Settlers（定住派）

| 項目 | 詳細 |
|------|----|
| 結成時期 | Day 4（Bastionが演説で呼びかけ） |
| 理念 | 帰還は不確実。VR内で持続可能な社会を作ることが最優先 |
| 組織構造 | Bastion を議長とする住民議会。多数決制 |
| 活動 | 農耕（ゲーム内の農業コンテンツを食料生産に転用）、建築、教育、治安維持 |
| Exodusとの関係 | 対立的だが暴力は行使しない。「帰還が成功するなら歓迎するが、失敗のリスクを全員に負わせるな」 |
| 課題 | 長期的にはVR内のリソース（サーバー計算力、ストレージ）の有限性が問題になる |

### FACT-003: Berserkers（暴徒）

| 項目 | 詳細 |
|------|----|
| 結成時期 | Day 3（最初の暴動から自然発生） |
| 理念 | なし。力がすべて。リソースは奪うもの |
| 組織構造 | Fenris を頂点とする軍事的ヒエラルキー |
| 活動 | 略奪、PK（プレイヤーキル）、領土拡大、「税」と称したアイテム徴収 |
| 支配地域 | Crimson Peaks 全域 + Obsidian Citadel の一部 |
| 弱点 | 内部の忠誠心は恐怖に基づく。Fenris がいなくなれば瓦解する |
| Exodusとの関係 | 敵対。ただしFenris は最終的に「帰還できるなら俺も帰る」と判断する可能性（OQ-CHAR-002） |

### FACT-004: The Awakened（覚醒者）

| 項目 | 詳細 |
|------|----|
| 結成時期 | Day 6（Oracleが最初の「啓示」を語る） |
| 理念 | 意識がデータになったことは「進化」であり「輪廻からの解放」。帰還は「退化」 |
| 教義 | VR世界は「真の世界」。Arcadia は楽園であり、現実世界こそが仮想 |
| 儀式 | Ethereal Gardens の浮遊島で「瞑想」（VR内睡眠＋NCE生成の幻視） |
| 危険性 | 信者が「帰還を妨害する」過激行動に出る可能性 |
| Exodusとの関係 | 当初は無関心 → 帰還計画が具体化すると対立 → 最終的にOracleが「選択の自由」を認める |

### FACT-005: Loyalists（運営擁護派）

| 項目 | 詳細 |
|------|----|
| 結成時期 | Day 2（ニュース流入直後） |
| 理念 | 運営は事件を解決しようとしている。内部からの独自行動は事態を悪化させる |
| 根拠 | 「外部の専門家のほうが確実に解決できる。素人がシステムをいじるのは危険」 |
| 活動 | 情報収集、RSS上での議論、Exodusの「過激な行動」への牽制 |
| 弱点 | 時間が経つほど「運営は何もしてくれない」という証拠が積み上がり、構成員が流出 |
| 崩壊 | Day 20頃に半数以上がExodusまたはSettlersに移行。Advocate は最後まで残る |

## 派閥間の力学

### 時系列変化

```
Day 0-2:  全員が混乱状態。派閥なし
Day 3:    暴動発生。恐怖を感じた集団が自然に固まる
Day 4-5:  Settlers, Exodus が正式結成
Day 6:    The Awakened が出現
Day 7-10: 領土争いが激化。Berserkers が Crimson Peaks を制圧
Day 11-15: 勢力均衡。各派閥が境界を確立
Day 16-20: Loyalists が縮小。Exodus に技術者が集中。**AdvocateがECHO経由でExodusの進捗情報を外部に漏洩**
Day 21-25: Exodus の進捗が可視化 → 他派閥からの合流が増加
Day 26-28: 最終決戦前。一時休戦の交渉
Day 29-30: 全派閥が（程度の差はあれ）帰還作戦に協力。**Advocateの裏切りが発覚**
```

### 資源配分の対立

| 資源 | 争奪状況 |
|------|---------|
| Arc（通貨） | Berserkers が略奪 → 経済混乱 → Settlers が物々交換制度を導入 |
| 食料アイテム | 農業可能なVerdant Expanse を Settlers が支配 → 他派閥は購入か略奪 |
| 武器/防具 | Crimson Peaksのダンジョンでドロップ → Berserkers が独占 |
| 情報 | RSS経由 → Gazette (`CHAR-011`) が中立的に管理しようと奮闘 |
| 保守APIアクセス | Exodus が独占 → 他派閥から「共有しろ」と圧力 |
