# 第七話　暗号解読

*Phase 4: 不信と崩壊 ─ EP-007 / VR内 Day 15-18*

---

## 1. トークン

　Day 15。Exodus Base Camp。

　蓮とFirewallは、保守APIの認証壁に正面からぶつかっていた。

「`MAINTENANCE_ADMIN`ロールのJWTトークンが必要だ。でも生成するにはSECRET KEYが要る──普通の手段じゃ取れない」

　Firewallがターミナルの画面を指差した。

「だが、監査ログに手がかりがある。v2.1.0-rc3のデプロイ時に、テスト用トークンが短期間だけ発行されていた形跡がある。そのトークンの有効期限は切れてるが──」

「署名アルゴリズムの解析から、KEYの候補を絞れる、と？」

「正確には、JWTのヘッダーからアルゴリズムとキーIDが読める。キーIDが分かれば、対応するKEYの格納場所を推測できる。……ただし、それを取得するには権限が──」

「ループしてるな」

「ループしてる」

　蓮は頭を抱えた。権限を得るにはトークンが要る。トークンを生成するにはKEYが要る。KEYを取得するには権限が要る。──鶏と卵。

「……別のアプローチを考えよう」

　Lynxが割り込んだ。

「あたしと技術チームでAPIの監査ログを総当たりする。過去のリクエストパターンから、認証バイパスの穴を探す。三十人で手分けすれば、三日で──」

「三十人？」

「Exodusのコアは十人だけど、技術に自信のあるテスターを追加で二十人集めた。ネットワーク班、暗号班、フロントエンド班──」

　蓮は驚いた。いつの間にExodusの協力者がそこまで増えていたのか。

「Bastionの定住派からも何人か来てるよ。『帰還は無理でも、技術的に面白い』って理由だけど。動機なんか何でもいいんでしょ？」

　──自分の言葉を返された。蓮は苦笑した。

## 2. Advocateの話のすり替え

　Day 16。

　Exodusが監査ログと格闘している間、RSSにAdvocateの投稿が上がった。

```
[RSS] #general
@advocate: 皆さんに進展を報告します。
　外部から法的手続きが進んでいます。
　国際仲裁裁判所がNH社に対する緊急裁定を
　検討していると複数の情報筋が伝えています。
　テスター救出に向けた法的枠組みが整いつつある。
```

　Lynxが蓮の横でRSSを読み、眉を吊り上げた。

「ちょっと待って。前は『専門家チームが動いてる』って言ってたよね。今度は『法的手続き』？」

「……話が変わってる」

「ていうか、進化してるよ。微妙にね。最初は『運営が対応する』。次に『外部で専門家が動いてる』。で今度は『法的手続き』。──毎回、もっともらしい理由に更新されてるけど、結局『待て』の結論は変わってない」

　蓮はNFCのニュースフィードを確認した。

　──国際仲裁裁判所の動きは──確かに記事がある。だが内容は「管轄権の議論が開始された」という段階で、「緊急裁定を検討」とは微妙にニュアンスが異なる。

「……Advocateは、善意で解釈を広げてるのか。それとも──」

「どっちにせよ、あたしたちの仕事は変わらないよ。Ren、トークンの件、進展あり」

## 3. 裏切り

　Day 17、深夜。

　Base CampでFirewallが叫んだ。

「──侵入者！」

　蓮が駆けつけると、ターミナルの一台が異常なアクセスログを吐き出していた。

「Base Campの座標データが外部に送信されてる──Exodusのメンバーの誰かが」

「何だと？」

　蓮は血の気が引いた。

「送信先は──Crimson Peaksのネットワークセグメント。Berserkersだ」

　Exodusメンバーの一人が──蓮たちの活動拠点の位置情報を、Fenris率いるBerserkersに漏洩していた。

　犯人は程なく特定された。ネットワーク班に参加していたテスターの一人。Fenrisに「帰還技術を渡せば見逃す」と脅されていたことを、取り調べで告白した。

「……Fenrisは、俺たちの技術が欲しかったのか」

「違う」Lynxが首を振った。「情報をエサにして、あたしたちを潰しに来るつもりだよ。Berserkersにとって、帰還派の存在は──秩序への脅威だから」

## 4. 襲撃

　Day 18、早朝。

　Fenrisが来た。

　Undercroft第三層の通路を、五人のBerserkersが縦列で突進してくる。先頭のFenrisは長剣を肩に担ぎ、足音を立てずに走っている。傭兵の走り方だ。

「総員、防衛態勢！」

　蓮が叫んだ。Base Campへの侵入経路は三カ所。うち二カ所はバリケードで塞いであったが、裏切り者が一カ所のパスワードを教えていた。

　Berserkersの攻撃は統率されていた。二人が前衛で盾持ちを崩し、Fenris自身が側面から突入する。蓮とFirewallが迎撃に出たが、FenrisのVR内戦闘能力は桁違いだった。

「──ッ！」

　蓮の短剣がFenrisの長剣に弾かれた。手首に衝撃が走り、武器がスピンしながら飛んでいく。

「……お前がExodusのリーダーか」

　Fenrisの声は低く、冷静だった。殺意というより──品定めの目つき。

「帰還を追求するのは結構だ。だが──その技術は俺が管理する方が効率的だ」

「……お前が管理して、どうする」

「交渉材料にする。外部との──あるいは内部との」

　蓮はFenrisの目を見た。

　──この男は、単なる暴漢ではない。何かを知っている。何かの計算で動いている。

　防衛戦は二十分続いた。Exodusメンバー三名が気絶に追い込まれ、アイテムの多くを失った。だがBase Campのターミナルだけは──Firewallが体を張って守り切った。

　Berserkersが撤退したのは、Lynxが予め仕込んでいたトラップ──Undercroft第二層の扉のロックを遠隔操作で閉じ、退路を脅かしたから──だった。

「……三名がダウン。ポーションゼロ。だが──」

　蓮はターミナルの前に座り、画面を確認した。

　Firewallが静かに言った。

「防衛中に、並行でトークン再署名のプロセスを走らせていた。──成功した」

```
> whoami
Role: MAINTENANCE_ADMIN
Session: valid
Permissions: [maintenance.read, maintenance.write, governance.read]
```

「`MAINTENANCE_ADMIN`……取れた」

　蓮は額をターミナルに押し当てた。

「Firewall……お前、いつの間に」

「襲撃が来た時点で、三日間の解析結果を流し込んだ。時間がなかったから、検証を省略して直接実行した。──博打だったが」

「……ありがとう」

　Firewallは黙って頷いた。

　画面に表示されたのは、Lethe Chainのガバナンスコントラクトの構造──

```

{
  "type": "MultiSigGovernance",
  "threshold": "5-of-7",
  "signers": [
    {"id": "SIGNER-001", "entity": "NH_CORP", "status": "VALID"},
    {"id": "SIGNER-002", "entity": "NH_CORP", "status": "VALID"},
    {"id": "SIGNER-003", "entity": "ETH_FOUNDATION", "status": "VALID"},

  "reason": "Insufficient valid signers (4/5 required minimum)"
```

　蓮は画面を凝視した。

「マルチシグ。5-of-7。七人の署名者のうち五人の署名が必要……だが──」

「COHEN_TRUSTのHSMが破壊されてる。INDEPENDENTは取締役会で議決権を剥奪」

　Lynxが背後から画面を覗き込み、息を呑んだ。

「つまり有効な署名者は五人。閾値ぎりぎりのはず──だけど──」

「`BLOCKED`。五人いるのに通らない？」

「いや、四人しか署名できてないって書いてある。五人目が──何らかの理由で署名を拒否？」

　Fenrisの捨て台詞が蓮の脳裏をよぎった。

「『サイバー攻撃……なるほどな』──あいつ、何を知ってるんだ？」

　蓮は画面を閉じ、天井を見上げた。

　ガバナンスの凍結。署名者の破壊。タイミングの「偶然」──

　──これは、本当に事故なのか？
