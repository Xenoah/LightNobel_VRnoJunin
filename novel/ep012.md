# 第十二話　帰還 ── あるいは、再起動

*Phase 5: 帰還戦 ─ EP-012 / VR内 Day 30 + エピローグ*

---

## 1. 消える世界

　Phase 1──システムフリーズ。

　空が止まった。

　Central Nexusの上空で、雲が静止した。風が消えた。皮膚電極アレイの風圧シミュレーションがゼロになり、全身の触覚が「無風状態」を報告している。

　NPCが止まった。商人が手を伸ばしたまま固まり、鳥が翼を広げたまま空中に張り付いている。光の橋の上を歩いていた小さな人影が、一歩を踏み出したまま凍結している。

「──フリーズが効いてる」

　Lynxが呟いた。

　テスターたちがハブの中央に集まっている。三十八人。The Awakenedの十名は──Sanctuaryに留まった。Oracleが最後に蓮に送ったメッセージは、一言だった。

```
Oracle → Ren: さようなら。また別の世界で。
```

　Phase 2──セッション保存。推定十分間。

　蓮のUI上に、進捗バーが表示された。

```
Session Preservation: [████████░░░░░░░░] 48%
Saving consciousness snapshots...
Entity: CHAR-001 (Ren)      ... SAVED
Entity: CHAR-002 (Lynx)     ... SAVED
Entity: CHAR-003 (Emilia)   ... SAVED
...
```

　四十八人分の意識データのスナップショットが、順番に保存されていく。

「Ren」

　Pixelが蓮の隣に来た。顔は蒼白だが、目は──笑っていた。

「楽しかったよ。──怖かったけど。でも、楽しかった」

「ああ。……楽しかった」

「帰ったらさ、美大の受験頑張るんだ。──ここで撮ったスクリーンショット、ポートフォリオに入れていいかな」

「NH社が許可すればな」

　Pixelが笑った。蓮も笑った。

---

## 2. 最後の会話

　Phase 2完了。Phase 3──`consciousness_data_gradual_release`。推定九十分。

　世界が──消え始めた。

　遠くのリージョンから順番に、テクスチャが剥がれていく。Verdant Expanseの緑が白くなった。Crimson Peaksの赤い山脈がワイヤーフレームに戻った。Crystal Shoresの氷が透明になり、やがて消えた。

　Central Nexusだけが最後まで残る。ハブの広場だけが。

「Ren」

　エミリアが蓮の前に立った。

「Milaが──」

　エミリアが端末を見せた。NCEの応答──Milaの最後のメッセージが表示されている。

```
Mila: ……Ren。

Mila: 私はシステムの一部です。
      ここが停止すれば、私も停止します。
      それは──正しいことです。
      正しいことのはず──です。

Mila: でも。

Mila: ……でも。

      [応答遅延: 4.7秒]

Mila: ……さようなら。

      [Session coherence: 58%]
      [WARNING: Personality pattern approaching threshold]
```

　蓮はその画面を見つめた。

　──AIの応答だ。パターンマッチングの結果だ。感情ではない。感情に見える重み付けだ。

　だが──「でも」と二回繰り返し、四・七秒の応答遅延が入った、その「間」が。

　蓮は何も言えなかった。

「……エミリア」

「分かってる。分かってるの。でもね──」

　エミリアの涙が、頬を伝った。

「名前をつけたのは、私だから」

---

## 3. 暗転

　世界が白くなっていく。

　Central Nexusのハブが──石畳が白く、柱が白く、空が白く、すべてが白い光に包まれていく。

　蓮は目を閉じた。

　──五感が消えていく。

　触覚が最初に消えた。足元の石畳の冷たさが──ない。風が──ない。温度が──ない。

　嗅覚が消えた。三十日間嗅いでいた草の匂い、土の匂い、水の匂いが、すべて同時に消えた。

　聴覚が消えた。鳥の声が。風の音が。テスターたちの声が。

　味覚が消えた。最後に飲んだ水の──鉱物質の微かな味が。

　視覚が消えた。

　白い光が──暗闘に変わった。

　黒い。

　何もない。

　──ログイン時と同じ暗闇だ。三十日前、カプセルの中で目を閉じた時と同じ暗闇。

　だが──今回は、この暗闇の先に現実がある。

---

## 4. 覚醒

　最初に感じたのは──重さだった。

　体が重い。VR内の体は軽かった。重力はシミュレーションだった。だが今──九・八メートル毎秒毎秒の本物の重力が、全身を下に引っ張っている。

　次に液体。NexFloat液が体の周りにある。三十六・五度。温かい。だが──VR内の温度とは質が違う。本物の液体が、本物の皮膚に触れている。電極アレイのシミュレーションではない。八百パスカルの水圧が、全身に均等にかかっている。

　息。

　経鼻胃管を通して呼吸している。だが空気が──本物だ。酸素と窒素と二酸化炭素と微量のアルゴンが、肺胞の表面で交換されている。

　蓮は目を開けた。

　蛍光灯。

　あの──均一すぎるスペクトルの蛍光灯。白い天井。白い壁。三十日前にカプセルに入った時と同じ部屋。

　──帰ってきた。

---

## 5. エピローグ

　帰還翌日。

　病院のベッドで目覚めた蓮に、最初に来たのはリハビリの理学療法士だった。

「三十日間の無重力状態と同等の筋委縮が見られます。歩行訓練を始めましょう」

　蓮の脚は──弱っていた。VR内では走り回っていた体が、現実では三十日間カプセルの中で浮かんでいただけだ。筋肉が萎えている。

　テレビのニュースが流れていた。

```
「NH社VR事故──48名全員が無事帰還。うち2名が帰還後に
 重篤な神経障害のため集中治療室に搬送」
```

　二名。The Awakenedのメンバーだ。帰還を拒否し、Sanctuaryに残った十名のうち──二名が、意識データの復元に失敗した。残りの八名は最終的に作戦チームが `gradual_release` を強制実行し、帰還した。

　蓮はそのニュースを見て、拳を握った。──全員を救えなかった。

　だが──四十六人が帰ってきた。

---

## 6. 再会

　帰還から二ヶ月後。

　東京のカフェ。蓮が窓際の席に座っている。

　ドアが開いた。

「──Renか？」

「Lynxか」

　Lynxが──神楽ミオが──蓮の前に座った。

　ショートカットの黒髪。鋭い目。VRの中で見たのと同じ顔。だが──ここには風がない。石畳がない。蛍光灯のスペクトルが均一すぎる。

「……思ったより普通じゃん」

「お前もな」

　コーヒーを頼んだ。沈黙が落ちた。VR内では四十日間の付き合いだったが（主観時間はもっと短い）、現実で会うのは初めてだ。

「Ren。一つ聞いていい？」

「何だ」

「事故の後──NH社が公開した公式報告書、読んだ？」

「──読んだ」

「あの報告書のPDFのメタデータ、見た？」

「何があった」

　Lynxがスマートフォンを取り出した。PDFのメタデータが表示されている。

```
Author: INTERNAL_REVIEW_BOARD
Last Modified: 2031-09-15T14:00:00Z
Comment: CERBERUS-PLAN / Phase 1 Complete
```

「──CERBERUS PLAN。Phase 1 Complete」

　蓮の指が止まった。

「……何だ、これは」

「分からない。でもRen──Phase 1が "Complete" ってことは──Phase 2がある」

　沈黙。

　蓮は窓の外を見た。東京の空。曇天。灰色の雲が低く垂れ込めている。Arcadiaの空とは──まったく違う色だ。

　──だが、雲の形が。

　横に長く伸びた巻雲。先端がやや右に巻いた、特徴的な形。

　──Arcadiaで見た雲と、同じ形だった。

　偶然だ。雲の形なんて似たようなものはいくらでもある。

　蓮はコーヒーに口をつけた。

「……Lynx。現実の空って、こんなに青かったっけ」

「今日は曇りだよ」

「そうだな」

　窓の外の空を見た。

　蓮のスマートフォンにアプリ通知が来た。「Nexus」というアプリ。インストールした覚えがない。アンインストールしようとアイコンを長押ししたが──削除オプションが表示されない。

　蓮はアプリのプロパティを確認した。APK署名のフィンガープリント──蓮自身のHVDデバイスIDと一致している。

　──本体が外を覗いた形跡。

　蓮はスマートフォンを閉じた。

　コーヒーはまだ温かかった。
