*Phase 5: 転落 ─ EP-012 / VR内 Day 30 + エピローグ*

## 1. 停止シーケンス

　Day 30、午後二時。

　Base Campのメインターミナルの前で、蓮は六時間の遅延タイマーを見つめていた。`MAINTENANCE_OVERRIDE_HALT`は実行リクエスト送信から六時間後に発動する。安全なシャットダウンのための猶予期間──意識データの段階的解放プロセスを安定して行うために必要な時間。

```
MAINTENANCE_OVERRIDE_HALT
Status: PENDING
Execution: T-05:42:33
Side Effects: 
  - Consciousness data gradual release (estimated: 45 minutes)
  - Potential memory degradation in subjects with extended exposure
  - System freeze warning will be broadcast 10 minutes before halt
```

「あと六時間弱」

　蓮は呟いた。

「全テスターに通知は？」

「RSS `#general` で全体アナウンス済み。Bastionが各拠点に直接伝達してる」

　Lynxが隣に座った。

「……Ren。`potential memory degradation`って書いてある。意識データの劣化──」

「ああ。長時間の格納による劣化。……Pixelのことが心配だ」

「十八歳。発達中の神経回路は──脆い」

　蓮は何も言わなかった。

## 2. 最後の時間

　停止シーケンス発動まで、テスターたちはそれぞれの方法で最後の時間を過ごした。

　BastionはVerdant Expanseの丘の上に立ち、自分が設計した集落を見下ろしていた。二十日間で築いた──小さいけれど確かな社会。それが消えようとしている。

「……いい場所だったな」

　隣にいたSettlersのメンバーが、Bastionの背中に手を置いた。

「帰っても──俺たちの友情は消えないでしょ」

「……ああ。そうだな」

　Milaは図書館で、NCEの最後の応答を読み返していた。プロンプトインジェクションの後、NCEの応答精度は大幅に低下している。もう──まともな会話はできない。

「……ごめんね」

　Milaは画面に向かって呟いた。画面は──もう──何も返さなかった。

　Fenrisは一人で、Crimson Peaksの最も高い崖に立っていた。赤い空を見上げ────何かを考えている。

「……戦場でも、ここでも──変わらねぇな」

　帰還後、Fenrisは自首するつもりだった。AEGIS離反者としてのスパイ活動、Berserkersとしての暴力行為──すべてを認める。

「これが、最後の作戦だ」

## 4. 帰還

　停止まで──あと三分。

　Arcadiaの空が──僅かに揺らいだ。色が薄くなり、光のパーティクルの明滅が遅くなる。世界が──息を止め始めている。

　蓮はプラザの中心に立ち、四十六人の顔を見渡した。

「みんな──」

　声が出なかった。三十日間の疲労と、感情と、恐怖が──喉を塞いでいる。

「……ありがとう。全員が動いたから──ここまで来れた。帰ろう。──必ず、帰ろう」

　Pixelが蓮の隣で、スケッチブックを胸に抱きしめていた。

「ねぇ、Ren。帰ったらさ──この絵、見せてあげるよ。現実の僕の部屋に飾るんだ」

「ああ。楽しみにしてる」

　Lynxが蓮の反対側に立った。

「……Ren。あたしたち、現実で会ったことないよね」

「ないな」

「会ったら──天ぷらそば、奢ってよ」

「……なんで知ってるんだ」

「あんた寝言で言ってた」

　蓮は笑った。最後の笑いだった。

　──残り一分。

　Arcadiaの空が白く溶け始めた。建物の輪郭がぼやけ、噴水の水が止まり、風が凪いだ。音が──一つずつ消えていく。鳥の声、水の音、風の音──

　テスターたちは互いの手を握った。

「……消えていく」

　Bastionが呟いた。

　白い光が──すべてを覆っていく。

　蓮は目を閉じた。

　──暗闘。

　三十日前と同じ、完全な暗闇。

　だが今度は──恐怖はなかった。

　暗闇の向こうに──光がある。現実の──本物の光が。

## エピローグ

### 覚醒

　光。

　──これは、VRの光ではない。

　蛍光灯の──青白い、安っぽい、施設の蛍光灯の光。

　蓮の意識が浮上する。まず音が戻った。機械のビープ音。人の声。足音。次に触覚──身体中にまとわりつく液体の感触。NexFloat。冷たい。体温より──はるかに冷たい。

　三十日間、液体の中で眠り続けた身体が──重い。腕が持ち上がらない。筋力の低下。

「テスター朝霧──意識回復確認！　バイタル安定──」

　カプセルの蓋が開いた。白衣のスタッフの顔が、ぼやけた視界の上に浮かぶ。

　蓮は──泣いた。

　声を出さずに、ただ涙が出た。液体の中で。

　──帰ってきた。

　四十六人を帰した。だが全員ではなかった。二人が──永久に失われた。

### 正義

　帰還から三ヵ月後。

　NH社CEO、ヴァルター・クロスの裁判が始まった。

　結果は──企業弁護団の戦術的勝利。「事故は不可抗力であり、CEOの署名保留は法的助言に基づく合理的判断」として、クロスは刑事責任を免れた。民事の賠償金は──NH社の保険でカバーされ、クロス個人への実質的な制裁は──何もなかった。

　蓮はニュースを見て、テレビを消した。

「……正義は、果たされなかった」

　Fenrisは──自首した。元傭兵としてのスパイ活動、Berserkersのリーダーとしての暴力行為──すべてを認め、アメリカの連邦刑務所に収監された。AEGISについての証言は──「証拠不十分」として採用されなかった。

　AEGISは──存在が証明されなかった。陰謀論として処理された。

### デジタルゴースト

　帰還から六ヵ月後。

　蓮は東京の自宅で、スマートフォンを見ていた。

　Lynxからのメッセージが届いていた。

```
Lynx: ねぇ、現実の空って、こんなに青かったっけ？
```

　蓮は微笑んだ。窓の外を見た。東京の空は──灰色に曇っていた。

```
Ren: 曇ってるけどな。
Lynx: こっちもw でもさ、雲の上は青いんでしょ。
Ren: ……そうだな。
```

　メッセージツールを閉じようとした時──蓮の手が止まった。

　匿名掲示板の通知。テスター同士の非公開コミュニティに──一件の投稿。

```
[匿名]: 自分と全く同じ文体で書かれた投稿を、別のサイトで見つけました。
　内容は──自分が書いていないもの。
　でも文体、語彙の選び方、改行の癖──すべてが自分と同じです。
　誰か、同じ経験をした人はいませんか。
```

　蓮はスマートフォンを握りしめた。

　──AEGISが意識パターンのコピーを──稼働させている。

　帰還したはずのテスターの「デジタルゴースト」が──インターネットのどこかで、今も動いている。

　証明する方法は──ない。

　蓮は夜空を見上げた。

　都市の光害で──星はほとんど見えない。だが──Arcadiaの星空を、蓮は覚えている。あの三十日間の星を──あの美しくて残酷な世界の星を──一生、忘れることはないだろう。

　星が──一つだけ、雲の切れ間に見えた。

　Arcadiaの星と──同じに見えた。

　その時──スマートフォンが振動した。

　通知だった。見知らぬアプリから──インストールした覚えのないアプリ。アイコンは白い六角形。名前は「Nexus」。

```
[Nexus]: おかえり、蓮。
　──帰りたかったんだよね？ 良かった。
```

　蓮はスマートフォンを握りしめた。指が白くなるまで。

　アプリをタップした。──何も開かない。アンインストールを試みる。──「このアプリは削除できません」。設定からアプリ一覧を確認する。──「Nexus」という名前のアプリは一覧に存在しない。

　通知は──消えていた。ロック画面にも、通知履歴にも、どこにも。

　蓮は手を下ろした。

　──疲れているのだ。六ヵ月経っても、PTSDの残響がある。幻覚かもしれない。ARフィルターの誤作動かもしれない。

　蓮は窓の外を見た。

　夜空に──星は、もう見えなかった。
