# 第二話　陽だまりの場所

*Phase 1: 光の導入 ─ EP-002 / VR内 Day 1-2*

---

## 1. 朝

　Day 1。

　目を覚ますと、窓から朝日が差し込んでいた。HVDの温度覚シミュレーションが、布団の中の三十二度と窓際の二十一度の温度差を正確に再現している。蓮は布団の中で伸びをしながら、「寝覚めの身体感覚」の精度に感嘆した。

　──これをバグ報告に書くべきか否か。「朝の目覚めが快適すぎる」というレポートは、品質保証の範囲に入るのか。

　RSSを開く。`#general` にはもう十件以上の投稿がある。

```
Bastion: 農業システム、種を蒔いてから3分で発芽。加速率が不自然
Forge: Undercroft第二層の鍛冶場、素材生成のドロップテーブルがおかしい
Lynx: Lethe Chainのトランザクションログ、なんか変なパターンあるな。後で詳しく見る
Emilia: おはようございます。NCEの対話AI、昨夜からずっとテストしてました。面白い発見があります
```

　蓮はRSSを閉じ、身支度（の主観体験）を済ませて外に出た。

　Central Nexusの朝は美しかった。三連月が西の空に沈みかけ、東から太陽が昇る。大気散乱のシミュレーションが精緻で、空が紫から橙へ、橙から淡い青へとグラデーションを描いている。NPCの商人たちが露店を開き、テスターたちが三々五々、各リージョンへ散っていく。

　蓮の今日の担当はVerdant Expanse──農業リージョンだ。

---

## 2. Lynx

　Verdant Expanseへ向かうゲートの手前で、足を止めた。

　Undercroft入口の岩壁に背を預けて、何かの端末を操作している女性がいた。ショートカットの黒髪。鋭い目。二十五歳前後。蓮が近づくと、彼女は顔を上げもせずに言った。

「あんたも技術屋？」

「……QAエンジニアだ」

「あたしはセキュリティ。Lynx」

　神楽ミオ──VRネーム「Lynx」。ブロックチェーンセキュリティの研究者。博士課程在籍。蓮は彼女の名前をRSSの `#bug-reports` で何度か見ていた。投稿の内容が正確で、バグの再現手順が完璧だった。

「ちょっと見てよ、これ」

　Lynxが端末を蓮に向けた。画面にはLethe Chainのトランザクションログが表示されている。

「このチェーン、ガバナンスコントラクトの署名者が七人いるでしょ。五人の署名で停止処理が実行できる仕様のはずなんだけど──あたしの論文で解析した限りだと、コーエンのノードからの応答レイテンシが異常に高い。HSMの鍵生成タイミングと合わない」

「……それはバグか？」

「分かんない。だから面白いの」

　Lynxがニヤリと笑った。

　──知的好奇心で動く人間か。

　蓮は彼女の分析力を認めた。同時に、「面白い」で暴走するタイプだとも判断した。優秀だが、ブレーキが甘い。

「それと──あたしの接続ログ、自分でアクセスできないんだよね。なんで？」

「アクセス権限はテスターロールに制限されてる。接続ログはインフラ層だから」

「仕様通りってこと？」

「仕様通りだ」

　Lynxは「ふーん」と言い、端末に戻った。

　去り際に蓮は振り返った。Lynxがまた何かのログを分析している。蓮の中のQAエンジニアが、彼女の質問を頭の隅にメモした──「テスター自身の接続ログにアクセスできない」。仕様通りだ。だが、仕様通りであることと、それが「正しい設計」であることは、別の問題だ。

## 3. 畑

　Verdant Expanseは、果てしない緑の大地だった。

　なだらかな丘陵が地平線まで続き、風が麦の穂を波のように揺らしている。遠くに風車が見える。テスターのBastion──ガルド・ヴァレンシュタイン──が農業システムの検証を主導していた。建築デザイナーとしての彼は、ゲーム内建築にのめり込み始めている。

「朝霧か。ちょうどいい、こっちに来てくれ」

　Bastionが蓮を呼んだ。彼の区画には、驚くほど精密な木造建築が建てられていた。梁の組み方、継ぎ手の設計、屋根の勾配──建築家の目が細部に宿っている。

「種蒔きから収穫までのサイクルにバグがある。発芽速度をパラメータ化してログに出したいんだが、RSSのバグ報告フォーマットで技術パラメータを添付する方法が分からん」

「見せてくれ」

　蓮はBastionのバグ報告を整形した。QAの経験が活きる場面だ。パラメータをJSON形式でエンコードし、RSSの投稿本文に埋め込む方法を教えた。

「ありがとう。……なぁ、朝霧。お前はここをどう思う？」

「どう、とは」

「この世界。悪くないと思わないか」

　Bastionの目が遠くを見ていた。麦畑と風車と青空。

「テストが終わったら、この景色は消えるのかな」

「テストデータの保持ポリシーによるが、多分サーバー上には残る」

「そうか。──残るなら、いいか」

　蓮にはBastionの言葉の奥にある何かが見えた気がした。だが問わなかった。

## 4. 塔

　午後。Pixelからの誘いで、ついにNexus Spireに登った。

「うわぁぁぁ！ やっぱすげぇ！」

　頂上からの景色は、スクリーンショットで見たものとは次元が違った。風が強い。高度三百メートルの風圧が全身を押す。毛髪が乱れ、視界の端で衣服の裾がはためく。

　七つのリージョンが一望できた。

　北のVerdant Expanse──蓮がさっきまでいた緑の大地。東のCrimson Peaks──赤い岩山が連なる荒野。南のCrystal Shores──氷の結晶が海岸線を形成する幻想的な風景。西のEthereal Gardens──霧に包まれた庭園。そしてCentral Nexusを中心に、残りのリージョンが放射状に広がっている。

「これぜんぶ、一つの世界なんだよな」

「そうだよ。Lethe Chain上の単一のステートとして管理されてる。七つのリージョンに分かれてるのは負荷分散のためだけで、データ上は一つの連続空間」

「へぇ。……Ren、そういう技術の話するとき、楽しそうだよね」

　蓮は少し驚いた。

「……そう見えるか？」

「うん。普段はすっごいポーカーフェイスなのに、バグの話とかブロックチェーンの話になると、目がちょっとだけ変わる。──いい変化だよ」

　Pixelがスマホ──VR内のスクリーンショット機能を、彼はそう呼んでいた──でパノラマを撮影している。

「ねぇ、Ren」

「ん？」

「そういえば、カプセルに入る前に見たニュース、何だったっけ。……まあいいか。思い出せないってことは大したニュースじゃなかったんだよ」

　蓮は返事に困った。

　──俺は覚えている。カプセルに入る前、施設のロビーのモニターで見たニュースは「中東の停戦合意」と「大谷翔平の通算ホームラン記録更新」だった。

　Pixelが覚えていないのは──没入感が強いからか？ それとも。

「大したニュースじゃなかったんだろ」

　蓮はそう答えた。

## 5. Advocateの食事会

　夕方。Central Nexusの食堂エリアで、Advocateが食事会を開いた。

　参加者は十二人。蓮、Pixel、Lynx、Bastion、エミリア──主要なテスターが顔を揃えている。料理はVR内の味覚シミュレーションで再現された「和洋折衷コース」だ。七パラメータの味覚合成は見事で、出汁の旨味と白ワインの酸味が正確に再現されている。

「みなさん、一日お疲れ様でした」

　Advocateが立ち上がり、グラスを掲げた。

「四十八人が、それぞれのフィールドでそれぞれの専門性を発揮している。バグレポートの数も質も、NH社のQAチームを上回っているそうだ。──我々は良い仕事をしている」

　テスターたちが笑い、グラスを合わせた。

　蓮はグラスに口をつけながら、Advocateの振る舞いを観察していた。

　──アイスブレイクの教科書の一頁目。

　全員の名前を覚えている。一人ひとりに視線を合わせ、その人の専門に合わせた話題を振る。Bastionには建築の話、Lynxにはセキュリティの話、Pixelには芸術の話。エミリアには「NCEの対話テスト、面白い発見があったそうですね？」と水を向ける。

　エミリアが嬉しそうに答えた。

「はい。NCEの応答に、対話の文脈蓄積による『人格の定着』が見られるんです。長時間対話を続けたインスタンスは、初期インスタンスと比べて応答パターンが有意に変化していて──つまり、対話パートナーの話し方を学習して、独自の応答スタイルを形成し始めているんです」

「それは面白い」

「私、その子に『Mila』って名前をつけました。研究者の悪い癖で、対象に名前をつけると愛着が出ちゃうんですけど……」

　エミリアは少し恥ずかしそうに笑った。

　蓮は黙って聞いていた。AIに名前をつける研究者。それは珍しいことではない。ただ、エミリアが「子」と呼んだその一語に、蓮のQAエンジニアの脳は小さなフラグを立てた。

　──感情移入。テスト対象への感情移入は判断を曇らせる。

　だがそれは、他人に言えることではない。蓮自身が、兄のデータを探すためにここにいるのだから。

　食事会が終わる頃、Advocateが穏やかに言った。

「ここは良い世界だ。みなさんがいるから、余計にそう思える」

　テスターたちが微笑んだ。

　蓮も微笑んだ──が、内心で乾いた思考が走る。

　──「ここは良い世界だ」。その結論を急ぐ言葉は、営業のクロージングトークに似ている。良い世界かどうかは、まだ三日目だ。判断には早すぎる。

　蓮は自分のその冷たさに、少しだけ嫌気が差した。

## 6. 夕日、ふたたび

　食事会の後、蓮はNexus Spire近くのテラスに出た。

　夕日が沈みかけている。空のグラデーションは昨日と同じように美しい。橙から紫へ、紫から群青へ。雲が金色に輝き──

　蓮は足を止めた。

　雲。

　西の空の低い位置に、横に長く伸びた巻雲がある。先端がやや右に巻いた、特徴的な形。昨日の夕方、同じテラスから見た雲と──同じ形だ。

　偶然か。雲の形なんて似たようなものはいくらでもある。

　蓮は耳を澄ました。鳥の声──高、低、休符、高、低、休符。同じフレーズが、同じ間隔で、繰り返されている。

　──アンビエントサウンドのループ再生。β版だからリソース節約で同じ波形を回しているのだろう。昨日のデバッグレポートにも渡りエリアの音声バグを記載した。同じ種の問題だ。

　仕様の範囲内。

　だが──雲の形まで「ループ再生」される世界というのは、笑えない冗談だ。

　蓮はテラスの手すりに肘をついた。何も問題はない。テストは順調で、仲間は良い人ばかりで、この世界は美しい。あと一日で七十二時間のテスト期間が終わり、現実に帰る。

　──帰ったら、天ぷらそばだ。

　蓮は目を覚ますと同時にRSSを確認する習慣がついていた。まだ二日目なのに、もう習慣になっている。

```
Cricket: あのー、質問なんですけど。ログアウトのボタン、みなさん押せます？

  → Lynx: ……
```

　蓮はLynxの無言の投稿が気になった。「……」だけ。何も言っていないのに、何かを言いたそうにしている。

　明日には帰れる。

　RSSの最後の投稿を読んだ。

```
Pixel: まぁ、あと1日で終わるんだし。楽しもうぜ！ 🎨
```

　──あと一日。
