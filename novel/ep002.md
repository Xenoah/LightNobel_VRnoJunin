# 第二話　陽だまりの場所

*Phase 1: 光の導入 ─ EP-002 / VR内 Day 1-2*

---

## 1. 畑

　VR内二日目──Day 1の朝。

　蓮はVerdant Expanseの広大な草原の中に立っていた。視界の隅々まで、薄緑色の草が風に波打っている。遠景には紫色の花畑が霞み、頭上には白い積雲が悠然と流れている。

　足元では、蓮が三十分前に植えた仮想作物の苗が、もう根を張り始めていた。

「……成長速度パラメータ、現実の約八倍か。農業シミュレーションとしてはアリだが、リアリティ調整が必要だな」

　蓮はデバッグレポートを音声入力する。テスターとしての任務は多岐にわたるが、蓮の担当はシステム全般のQA。農業パートは専門外だが、「異常な成長速度を報告する」くらいのことは誰でもできる。

　しかし──正直、気分はいい。

　土を触る感覚がある。掘り返した土の湿った匂いがある。掌に泥がつき、それを拭う布の繊維まで感じる。VR内で農業をするのが「楽しい」とは思わなかった。だが五感で体験すると、それは確かに「作業」ではなく「体験」になる。

「おー、蓮くんじゃん。ここで何してんの？」

　声に振り向くと、見覚えのない赤髪のアバターが歩いてきた。メタリックなゴーグルを額に引っかけ、腕を組んでニヤリと笑っている。背は蓮より低いが、存在感がやけに大きい。

「……誰だ？」

「Lynx。よろしく、Ren」

　プロフィールが開く。セキュリティ研究者。大学院博士課程。ブロックチェーンセキュリティ。

「あんた、CentralのRSSで見たよ。デバッグエリアでLv差六のモブ倒したって書いてた人でしょ。技術屋なら話が合うかも」

「……RSSの投稿、俺はしてないが」

「Pixelが書いてた。『Renっていうすごい人がモンスター一人で倒した！』って。あの子、拡声器みたいだね」

　蓮は思わず苦笑した。

　Lynxは腕を組んだまま、草原の向こうに目を遣った。

「ねぇ、あんたも技術屋なら分かると思うけどさ──Undercroft、入ったことある？」

「入口付近だけだ。デバッグ任務で」

「あそこ、面白いのよ。元々は開発用のテストフロアがあったはずなんだけど、β版でアクセスが封鎖されてる。……でも封鎖の仕方が雑なの。壁の当たり判定にガバがあって、すり抜けられそうなポイントが二カ所」

「……それはバグレポートで書くべき内容じゃないか？」

「書くよ？　書いた後で入るの。順番の問題」

　ふてぶてしい笑みだった。蓮は「こいつは面倒なタイプだ」と思った。だが同時に、技術的な嗅覚が鋭い人間だとも分かる。βテスターにセキュリティ研究者がいるのは自然だが、Lynxの好奇心は──少し危ういものを含んでいた。

## 2. 塔の上

「Ren！　一緒に来てよ！」

　午前の任務を終え、Central Nexusに戻った蓮を待ち構えていたのはPixelだった。両手を大きく広げ、満面の笑みで蓮を引っ張りにかかる。

「Babel Towerのてっぺんまで行こう！　絶景ポイントらしいよ！」

「……テスト任務は？」

「午後のシフトまで自由時間！　ほらほら行こう！」

　蓮は断る理由を見つけられなかった。

　Babel Tower──Central Nexusの中心にそびえる巨大な塔。高さは公式データで二百メートル以上。内部には螺旋階段とRSS端末の集約拠点、テスター議会場が設けられている。だが最上階の展望テラスは「テスト対象外」として、まだほとんどのテスターが訪れていないらしい。

　螺旋階段を登る。VR内では体力ゲージが緩やかに減少し、息が切れる演出もある。Pixelは平気な顔で駆け上がり、蓮はやや遅れてついていく。

「Pixel、速すぎる」

「えー、Renしょぼくない？」

「レベルの問題じゃなく体力の仕様の問題だろ」

　だが──久しぶりに、誰かと一緒に「遊ぶ」感覚があった。仕事でもなく、検証でもなく。ただ階段を登り、息を切らし、笑いながら屋上を目指す。

　十数分後、展望テラスに辿り着いた。

「──うわあ」

　Pixelの声が、風に飛ばされた。

　眼下に、Arcadiaの全景が広がっていた。

　七つのリージョンが地平線の彼方まで連なり、それぞれの地形が独自の色彩で光を纏っている。Verdant Expanseの緑、Crimson Peaksの赤褐色、Azure Depthsの深い藍、Obsidian Citadelの黒曜石、Ethereal Gardensの白と金──

　雲海の上に浮かぶCentral Nexusから見下ろすと、それはまるで巨大な宝石箱を開いたようだった。

「やばい……やばいやばいやばい」

　Pixelはスケッチブックを開き、手が止まった。どこから描けばいいか分からないのだ。

「……こういう瞬間に、カメラじゃなくて自分の目で見られるの、すごいことだと思わない？」

「ああ」

　蓮は素直にそう答えた。

　二人は黙って、しばらくその景色を眺めていた。風が吹き、蓮の黒い髪とPixelの蜂蜜色の髪を揺らした。

## 3. Advocateの食事会

　夕刻──宿舎棟の共有ダイニングで、食事会が開かれた。

　発案者はAdvocateだった。

「せっかく四十八人も集まっているんだ。顔を合わせて話す機会を作りたいと思ってね」

　長テーブルには十五人ほどのテスターが集まっている。VR内の食事は「主観体験」だ。栄養は現実のカプセル内で経鼻胃管から供給されるが、VR内では味覚・嗅覚・触覚がフルに再現された食事を楽しめる。パンを千切る感触、スープの温度、焼いた肉の匂い──すべてが本物のように感じられる。

「このパン、本物より美味しい気がする……」

　Milaが小さな声で言い、隣のテスターが笑った。

「そりゃフィルター通してるからね。嫌な食感とか臭みは消してるはず」

　Advocateがワイングラスを掲げた。VR内のワインにアルコールの効果はない──酩酊シミュレーションはβ版では無効化されている──が、グラスを合わせる「カチン」という音は心地よく響いた。

「ここにいるのは、世界中から選ばれた四十八人だ」

　Advocateの声は落ち着いている。大人数の前でも臆さない。自然なリーダーシップ──というより、「この人がいると場が安定する」という種類のカリスマがある。

「技術者、研究者、デザイナー、アーティスト。それぞれのプロが集まっている。この三日間で、人類が初めて経験するフルダイブVRの品質を、俺たちの手で決める。……それって、すごいことだと思わないか？」

　テスターたちが頷く。笑顔がある。興奮と誇りがある。

「ここは良い世界だ」

　Advocateはそう締め括り、穏やかに微笑んだ。

　蓮はグラスに口をつけながら、その微笑みを見ていた。

　──何の違和感もない。

　ただ「いい人だ」とだけ思った。

## 4. 夜のNPC

　食事会の後、蓮はMilaと偶然すれ違った。

　Central Nexus南端の石橋の上。Milaは道端のNPCと対話している最中だった。花売りの老婆──スクリプトNPCだと思われるが、Milaの表情は真剣だ。

「……それで、あなたはこの花をいつから売っているんですか？」

　NPCが応答する。

「さあねぇ、もう何十年もになるかしら。この街ができた頃からだよ。……えっと、街ができたのは──」

　NPCが一瞬、不自然に停止した。長い沈黙。そして──

「──ごめんなさいね、おばあちゃん、ちょっとぼんやりしちゃって」

　Milaの目が光った。

「Ren、今の見ました？」

「……見た。NPCが応答に詰まった？」

「詰まっただけじゃないです。回復のフレーズが文脈に合ってる。通常のスクリプトNPCなら、タイムアウト後にランダムなフォールバック台詞を返すはずです。でもこのNPCは──」

「自分で誤魔化した」

「そう。推論してます。これ、普通のスクリプトじゃない。かなり大きな言語モデルが動いてると思います」

　Milaの頬が薄く紅潮している。NLP研究者にとって、これは金脈のような発見なのだろう。

「面白いですね」

　Milaは小さく呟いた。笑顔だった。純粋に、学術的好奇心に満ちた顔。

「……面白い、だけ？」

「はい。今のところは」

　蓮にはその「今のところは」の響きに何かが含まれているように聞こえたが、それ以上は追わなかった。

## 5. 夕日、ふたたび

　Day 2の夕方。

　蓮は再びBabel Towerの展望テラスにいた。今度は一人で──のつもりだったが、やはりPixelがいた。

「あ、Ren。また来た」

「……やたら行動パターンが被るな」

「ここが一番きれいなんだもん」

　Pixelはスケッチブックに色を重ねている。昨日描いた夕景に、今日の光の変化を加筆しているようだ。

　二人で西の空を見つめた。今日の夕日は昨日より赤い。雲の形も違う。Arcadiaの気象シミュレーションは日ごとに変化するらしく、飽きることがない。

「ねぇ、Ren」

「ん？」

「そういえばさ、カプセルに入る前に見たニュース、何だったっけ」

「……ニュース？」

「うん。施設のエントランスのモニターで、なんか流れてたじゃん。見たはずなんだけど、内容が思い出せなくて。……まあいいか」

　Pixelはスケッチブックに目を戻した。

　蓮は少し考えた。

　──エントランスのモニター。確かにニュースが流れていた。何の内容だったか。……EU議会のVR規制法案の公聴会？　いや、それは先週だ。施設に入った日は──

　蓮自身も、すぐには思い出せなかった。

　──フルダイブの没入効果だろう。

「気にするな。接続切れば戻る」

「だよねー」

　Pixelは笑い、スケッチに集中した。

　蓮は夕空を見上げた。

　何も問題はない。テストは順調で、仲間は良い人ばかりで、この世界は美しい。あと一日で七十二時間のテスト期間が終わり、現実に帰る。

　──帰ったら、天ぷらそばだ。

　蓮はそう思い、穏やかに目を細めた。

## 6. 最初の投稿

　Day 2の夜。

　就寝前にRSS `#general` を確認する。いつもの賑やかなログが流れているが、最新の投稿を見て蓮は手を止めた。

```
[RSS] #general
@unknown_tester_039: ログアウトできない人いる？
　メニュー開いてもログアウトボタンが見当たらないんだけど
```

　投稿は三分前。リアクションはまだ少ない。

「……βテスト仕様だな」

　蓮は呟いた。そうだ。HVDのMark III β版には、テスト中のログアウト機能は実装されていない。七十二時間の接続期間が終了すれば、施設側がカプセルの接続を解除して覚醒させる。それがプロトコルだ。テスト開始前の説明会で、明確に伝えられている。

　リプライが一つだけついた。

```
@tester_pixel: ログアウトボタンはβだからまだないんだって！
　あと1日で終わるし、楽しもうぜー！ (ﾟ∀ﾟ)
```

　──まぁ、その通りだ。

　蓮は端末を閉じ、ベッドに横になった。

　窓の外には星空が広がっている。Central Nexusの夜景が水面に映り、静かに揺れている。

　あと一日。

　明日の最終日は、The Undercroftの奥まで探索して、兄さんのデータを──

　蓮の意識は、そこで自然に途切れた。眠りに落ちた。

　──RSS `#general` の、別の投稿を、蓮はまだ知らない。

```
[RSS] #general
@tester_017: まぁ、あと3日で終わるんだし。楽しもうぜ
```
