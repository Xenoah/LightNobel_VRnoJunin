## 1. 解読

　蓮とLynxは、ガバナンスコントラクトの全構造を解読する作業に没頭していた。

　`MAINTENANCE_ADMIN`ロールで取得できた情報は膨大だった。マルチシグの署名者リスト、凍結理由のトランザクションログ、各署名者のHSM（Hardware Security Module）のステータス──

「整理するよ」

　Lynxがホワイトボード──VR内のインタラクティブパネル──に図を描き始めた。

```
署名者7名:
  SIGNER-001 (NH社・クロスCEO)     → 有効。だが法的助言で署名保留中
  SIGNER-002 (NH社・ヴァイスCTO)     → 有効。署名意思あり
  SIGNER-003 (Ethereum財団)          → 有効。技術的には署名可能だが法的リスクを懸念
  SIGNER-004 (FDAC規制当局)          → 有効。署名プロセスに時間を要する
  SIGNER-005 (コーエン信託)           → HSM物理的に破壊。署名不能
  SIGNER-006 (独立理事アルサード)     → 取締役会で議決権剥奪。署名権限無効化
  SIGNER-007 (NH社・開発部門代表)     → 有効。署名意思あり
```

「有効な署名者は五人。閾値も五。──でも、署名が四つしか集まっていない」

「誰が欠けてる？」

「SIGNER-001──クロスCEO。企業弁護団の助言で署名を保留してる。『停止操作が失敗した場合の法的責任を問われる』って──」

　蓮は拳を握った。

「四十八人の命がかかってるのに──法的責任の心配か」

「企業ってそういうもんでしょ」LynxはLynxで声を抑えているが、怒りが滲んでいた。「でもRen、もっとヤバいのはこっち」

　Lynxが画面の別のセクションを指差した。

```
SIGNER-005 (コーエン信託):
  HSM破壊日時: 2031-07-16T03:22:14Z
  破壊原因: 物理的アクセスによる回路焼損
  調査状況: 「外部からの不正アクセス」として捜査中

SIGNER-006 (独立理事アルサード):
  議決権剥奪日時: 2031-07-17T16:45:00Z
  剥奪理由: 臨時取締役会による解任決議
  決議内容: 「利益相反の疑い」
```

「……コーエンのHSM破壊が七月十六日。アルサードの解任が七月十七日。事件発生が七月十八日──」

「三日連続。Ren、これ──」

「偶然にしては出来すぎていないか？」

　蓮とLynxは同時に同じ言葉を口にした。

## 2. 【現実カットイン②】

*── NH社本社 モニタリングルーム。2031年7月28日 09:00 JST ──*

　エレナ・ヴァイスは、自分の署名鍵を握りしめていた。

　小さなUSBデバイス──HSMクライアント。これを専用端末に接続し、パスフレーズを入力すれば、ガバナンスコントラクトに署名できる。ヴァイスの署名は二番目。三番目のEthereum財団、四番目のFDAC──彼らはいつでも署名できる。七番目の開発部門代表も。

　足りないのは──クロスCEOの署名だけだ。

「Dr. ヴァイス、FDAC側の弁護士との電話会談が──」

「あとにして」

　エレナは法務チームの報告書を読んでいた。クロスの企業弁護団が提出したもの。

```
「停止操作の実行に伴うリスク分析：
  1. 停止シーケンスの不備により意識データが破損した場合、
     NH社は刑事・民事の両面で重大な責任を負う
  2. 現時点で停止シーケンスの安全性は100%保証されていない
  3. 法的助言として、外部の独立検証機関による
     安全性確認を経てから署名することを推奨する」
```

「……独立検証機関」

　エレナは声を絞った。その検証に──何週間かかるのか。何ヶ月か。テスターたちのカプセルの物理限界は六十日。もう十日が過ぎた。残り五十日。

　──内部からの脱出を信じるしかない。

　エレナはモニターを見た。四十八基のカプセルのバイタルデータ。心拍。呼吸。脳波。一人一人が──まだ生きている。

　──お願い。どうか──内部で、誰かが動いて。

## 4. 情報公開の議論

　Day 21。Babel Tower議会場。

　蓮、Lynx、Bastion、Advocate、Gazette──主要な派閥のリーダーと情報管理者が集まった。

　議題は一つ。ガバナンス凍結の全貌をテスター全員に公開するかどうか。

「公開すべきだ。全員にこの状況を知る権利がある」

　蓮の声は静かだが、明確だった。

「反対だ」

　Bastionが立ち上がった。

「この情報は人々を絶望させるだけだ。『運営は止めたくても止められない』？　それをいま全員に伝えて、どうなる？　パニックが再発するだけだ」

「だが隠し続ければ──」

「隠すんじゃない。タイミングの問題だ。まだ帰還の具体策が見えていない段階でネガティブな情報だけ公開するのは──」

「具体策がないからこそ、全員の知恵を借りるべきだ」

　Bastionと蓮が睨み合う。Bastionの目が揺れた。──この男は悪意で隠そうとしているのではない。テスターたちの精神的安定を──彼なりの「社会設計者」としての視点で──守ろうとしている。

　Advocateも口を開いた。

「私も慎重派だ。公開するとしても、表現を選ぶべきだ。『運営が止められない』ではなく──」

「事実を歪めろと？」

「歪めるのではなく、文脈を添えるんだ。パニックを防ぐために──」

「いいえ」

　Gazetteが、静かに声を上げた。

　ジャーナリスト気質の情報管理者。普段は寡黙だが、今日は目に決意が宿っている。

「情報の受け手を──過小評価しないでください。テスターたちは大人です。事実を知り、判断する能力がある。──検閲は、受け手の尊厳を否定する行為です」

　沈黙が落ちた。

　Gazetteは続けた。

「公開します。全文。注釈と背景情報を添えて。──それが、私の用意した報道です」

## 5. 公開

　Day 22、正午。

　RSS `#news` に、Gazetteが編纂した報告書が全文公開された。

```
[RSS] #news / GAZETTE REPORT
──────────────────────
【ガバナンス凍結の真実】

1. Lethe Chainのガバナンスはマルチシグ 5-of-7 で管理されている
2. 署名者7名のうち、2名が事件前に無力化された
   - コーエン信託: HSMが物理的に破壊（2031-07-16）
   - 独立理事アルサード: 取締役会で議決権剥奪（2031-07-17）
3. 残る5名のうち、クロスCEOが法的リスクを理由に署名を保留中
4. 結果: 停止コマンドに必要な5署名が揃わず、システムは凍結状態
5. 外部からの強制停止は意識データ破損のリスクがあり、実行不可

結論: 運営は──止めたかったが、止められなかった。
──────────────────────
```

　RSS `#general` が──沈黙した。

　五分間、誰も何も書かなかった。

　やがて、一つの投稿が上がった。

```
[RSS] #general
@bastion_settlers: ……俺たちは、誰を恨めばいいんだ。
```

　蓮はBabel Tower展望テラスに一人で立ち、Arcadiaの空を見上げていた。

　空は──今日も、美しかった。

　誰の下にも、加害者は探しにくい。運営は止めたかった。署名者が二人消えた。一人は法的リスクで動けない。──悪意がどこにあるのか、見えない。

　見えない敵ほど、恐ろしいものはない。

　蓮はRSS端末をスクロールし続けた。──そして手が止まった。

　投稿タイムラインの最下部に──一件の投稿が埋もれていた。

```
[RSS] #general
@tester_049: なるほど。困った状況ですね。
　投稿: 2031-07-28T11:58:22+09:00
```

　蓮はその投稿を二度読んだ。

　テスターは四十八人だ。`@tester_049` は存在しない。

　投稿のタイムスタンプは──Gazetteの報告書が公開される二分前。まだ公開されていない情報に対して、存在しないアカウントが反応している。

　蓮はアカウント情報を検索した。──プロフィールが返る。名前なし。アバター画像なし。参加日──`Day 0`。

　──RSSのインデックスバグだ。テスト環境のダミーアカウントが残っているのだろう。

　蓮はそう結論づけた。だが──「困った状況ですね」という丁寧語の文体が、妙に人間らしくて、打ち消しきれなかった。
