# 第一話　ログイン ── 新世界の扉

*Phase 1: 光の導入 ─ EP-001 / VR内 Day 0*

---

## 1. カプセル

　施設の廊下は白かった。

　白すぎて、壁と天井の境目が溶けている。蛍光灯のスペクトルが均一すぎるのだ。QAエンジニアの脳が自動的にバグ報告の書式を組み立てる──「照明ユニット間のスペクトル偏差がIEC規格の基準値を下回っている。仕様外の均一性。優先度：低。人体影響：なし」。

　朝霧蓮（あさぎり れん）はその思考に自分で苦笑し、白衣のスタッフに続いて歩いた。

「朝霧さん、こちらです」

　名前を呼ばれて足を止める。扉の向こうに試験室が広がっていた。NH社（Nexus Horizons Inc.）の第三テスト施設。天井高四メートルのフロアに、HVDカプセルが六基並んでいる。

　HVD──Human Virtualization Device Mark III。

　全長二メートル二十センチ。幅九十センチ。高さ一メートル十センチ。強化アクリルの透明カバーに包まれたチタン合金のフレームが、蛍光灯の光を受けて鈍く輝いている。カプセルの内部はフッ素系不活性液体──NexFloat──で満たされており、体温と同じ三十六・五度に維持されている。

　人間を浮かべて、意識だけを別世界に送り出す機械だ。

「朝霧さん。外部QA契約の朝霧蓮さんで間違いありませんか」

「はい」

「βテスト期間は七十二時間──三日間です。カプセルの生命維持は経鼻胃管による栄養供給、カテーテルによる排泄管理、ペルチェ素子による温度制御です。緊急時の覚醒プロトコルは事前説明をご確認いただいた通りですが、要点だけ繰り返しますと──」

　蓮は説明を聞きながら、カプセルの継ぎ目を指先でなぞった。樹脂パッキンの圧着状態は良好。通気バルブの向きも仕様通りだ。QAエンジニアの習慣で、自分が入る装置の品質検査を無意識にやっている。

　──三日間。

　それだけの話だ。三日間テストして、バグを潰して、レポートを書いて、報酬をもらう。

　兄さんの足跡を探す。

　その思考は、蓮の中で一瞬だけ強く光って、すぐに棚の奥へ押し込まれた。兄──朝霧陽。NH社の初期テスターだった。HVD-MK2のテスト中に意識バインド不全で脳死。それがβテストに応募した本当の理由だ。

　だが今はまだ、その話をする相手がいない。

「……準備ができました」

　白衣のスタッフがうなずく。

　蓮はカプセルに横たわった。NexFloat液が体温と同じ温度で体を包む。浮力が体重を消す。経鼻胃管の感触は不快だが、すぐに神経I/Oの初期化シーケンスが始まり、末梢感覚が遠くなる。

　最後に見えたのは、天井の蛍光灯だった。

　均一な白──

　そして、暗転。

## 2. ログイン

　黒い。

　それしか認識できない時間が、一秒か十秒か続いた。

　最初に来たのは光だった。上方から射す一条の白光。蛍光灯の貧弱なスペクトルではない。太陽だ。可視光のフルスペクトルが網膜に飛び込んできた──HVDのレーザー走査が、現実の太陽を超える解像度で眼球の奥を叩いている。

　次に色彩。

　空が青い。均一な空色ではなく、天頂から地平線へのグラデーションが四百段階以上に分解されている。雲が立体的だ。二次元のテクスチャではなく、水蒸気密度の違いが光の散乱角を変えている。

　──これはテクスチャマッピングじゃない。レイトレーシングですらない。ボリュームレンダリングだ。雲の内部に光が入って散乱して出てくるまでの物理を全部計算している。

　風。

　左頬に触れた。皮膚電極アレイの五百十二点が、風圧と温度の勾配を同時に再現している。秒速二メートル。気温十八度。湿度は──体感で四十パーセント前後。乾いた、高地の風だ。

　匂い。

　鼻腔マイクロニードルが嗅覚受容体を直接刺激する。草の青い香り。土の湿り気。遠くの水面から運ばれる鉱物質の微香。二百五十六種の基本香気のうち、少なくとも四十種が同時に発火している。こんな複雑な嗅覚マッピングは、地球上のどんなVRにも存在しない。

　そして音。

　鳥の声が頭上を横切る。左耳から入って右耳へ抜ける。骨伝導と鼓膜振動の併用が、音場の定位を完璧に再現している。

　蓮は、足元を見た。

　石畳だった。白い石灰岩が精密に組まれ、その隙間から小さな苔が生えている。苔の上に朝露が一粒、光を受けて虹色に分光していた。

「──嘘だろ」

　声が出た。自分の声だ。声帯の振動がVRの物理エンジンに変換され、空気を伝播して自分の鼓膜に戻ってくるフィードバックループが正常に機能している。

　蓮は顔を上げた。

　浮遊都市だった。

　巨大な岩盤が空中に浮かんでいる。岩盤の上に白い建築群が積み重なり、塔が空に向かって伸びている。建物と建物の間を光の橋が繋ぎ、橋の上を小さな人影が歩いている。都市の外縁部からは滝が流れ落ち、下方の雲海に消えていく。滝の水しぶきが太陽光を受けて虹を作り──それが二重虹になっている。

　アルカディア。

　Nexus Horizons社が五年の歳月と百二十億ドルを投じて構築した次世代メタバース。七つのリージョンで構成される没入型仮想世界。その玄関口──Central Nexusが、蓮の目の前にあった。

　俺は今、ここにいる。

　体温がある。重力がある。風が吹いている。石畳の冷たさが靴底を通して足裏に伝わる。これはゲームではない。少なくとも「ゲーム」という言葉が持つすべての語感を超えている。

　──いい世界じゃないか。

　QAエンジニアとしての分析的な視線が、一瞬だけ完全に消えた。

## 3. テスターたち

　Central Nexusの中央広場は「ハブ」と呼ばれている。

　円形の広場の中心に情報端末──RSS（Resident Social System）のアクセスポイントが立っている。蓮が空中に手を伸ばすと、半透明のUIパネルが出現した。

```
[RSS v3.2.1-beta]
Welcome, Ren.
Channels: #general, #bug-reports, #team-alpha, ...
Active Testers: 48/48
```

　四十八人。全員がオンラインだ。

　蓮はRSSの `#general` チャンネルを開いた。テスターたちの投稿が流れている。

```
Pixel: うおおおおお！！ この景色やばくない！？ 写真撮りまくってる🎨
Lynx: Undercroft方面のNPC応答にレイテンシあり。バグ報告出した
Bastion: Verdant Expanseの農業システム、思ったより作り込まれてるな
Emilia: NCEの対話AIに面白いパターンを見つけました。報告は夕方までに
```

　蓮はスクロールを止めた。四十八人──少数精鋭のβテスター。全員がプロフェッショナルで、全員がこの世界を評価するために選ばれた。

「おっ、あんたもCentral Nexusから始めたの？」

　振り返ると、少年が立っていた。明るい茶髪。緑色の目。十八歳──βテスト参加者の最年少。

「僕、Pixel。よろしくね、えーっと──」

「Ren」

「Renかぁ。日本の人？　僕も半分日本なんだけど──それよりさ、ここ見てよ。この柱の影のレンダリング、距離によってソフトシャドウの半径がリアルタイムで変わるの。普通のレイトレだとこんな自然にならないよ」

　──美術学生か。

　蓮はPixelの分析に少し驚いた。陽気な少年に見えるが、視覚表現への感度が高い。

「バグ報告は `#bug-reports` に出せばいい」

「知ってるよ！　もう三件出した。──ねぇ、広場の向こうに塔があるでしょ。あのてっぺんまで行きたいんだけど、一緒に来ない？」

　蓮は塔を見上げた。Nexusの中心に立つ「バベルの塔」──公式名称はNexus Spire。高さ三百メートルの展望塔で、頂上からはArcadiaの七つのリージョンが一望できるらしい。

「……後でな。先にデバッグエリアを確認したい」

「りょうかい！　じゃあ後で塔の上で会おう！」

　Pixelが走り去る。足音がドップラー効果で遠ざかっていく。蓮はそれを聞きながら、無意識にバグ報告の書式を頭の中で整理していた。

　──足音の反響特性。石畳の材質による反射係数。距離減衰曲線。……正常。仕様通りだ。

　QAエンジニアの脳は、美しい世界の中でも止まらない。

## 4. The Undercroft

　デバッグエリアは「The Undercroft」と呼ばれている。

　Central Nexusの浮遊岩盤の下層部に広がる洞窟群で、βテスト期間中のバグ再現テスト用に開放されている。暗い。松明の光が壁面の鍾乳石に反射して、赤味がかった揺れる影を作っている。

　蓮はデバッグコンソールを開いた。

```
[Debug Area: The Undercroft / Sector B-7]
Active Entities: 12 (NPC-hostile)
Known Bugs: 3 (physics, pathfinding, drop-table)
Test Priority: Medium
```

　敵性NPC──つまりモンスターが十二体。蓮は戦闘用UIを確認した。

　このVRの戦闘システムは、現実の身体動作をそのままコマンドに変換する。剣を振れば、腕の軌道がそのまま攻撃判定になる。フットワークが回避になる。痛覚は最大九十五パーセント減衰されるが、「圧迫感」として残る──つまり、殴られれば痛くはないが、「何かに押された」感覚がある。

　洞窟の奥から低い唸り声が響いた。

　蓮は構えた。

　影の中から現れたのは、二メートル超の甲殻生物だった。六本の脚。頭部に複眼。背中に結晶質の突起が並び、松明の光を受けて赤く輝いている。デザインは見事だが──

「パスファインディングが壊れてるな」

　モンスターが柱に引っかかった。進行方向のノードが柱の背後に設定されているのに、衝突判定が先に発火して経路を再計算しない。典型的なA*のノードスキップバグだ。

　蓮はモンスターの攻撃範囲を観察しながら、バグ報告を音声入力した。

「Bug ID仮番。The Undercroft、Sector B-7。敵性NPC、種別：甲殻型。パスファインディングにおいて、障害物との衝突後に経路再計算が発火しない。再現率百パーセント。優先度：中。映像ログ添付」

　もう一体が背後から迫ってきた。蓮は身を低くして回避し、カウンターで剣を振った。刃が甲殻を叩き、「圧迫感」が手首に返る。ダメージ判定のエフェクトが散る。

　──戦闘システム自体は問題ない。ヒット判定、ダメージ計算、エフェクトレンダリング、すべて仕様範囲内だ。

　三十分で三体のバグを記録し、蓮はUndercroftを後にした。

## 5. 会合

　夕方、Central Nexusのハブに戻ると、テスターたちが集まっていた。

　四十八人中、三十人ほどがハブの広場にいる。残りはまだ各リージョンでテスト中だ。

「良いチームだな」

　声の主は、広場の端に立つ四十歳前後の男だった。端正な顔立ち。黒い髪を後ろに撫でつけ、落ち着いた物腰で周囲を見渡している。RSSのネームプレートには「**Advocate**」と表示されていた。

「四十八人。少数精鋭だ。全員が何かのプロフェッショナルで、全員がこの世界をより良くしたいと思っている。……いいじゃないか」

　Advocateはテスターたちの集まる広場を見渡し、目を細めた。

　蓮はグラスに口をつけながら、その微笑みを見ていた。

　──できすぎている。

　QAエンジニアの職業病だ。「信頼できると感じる」のと「信頼できる」のは別の命題だ。声のトーン、視線の合わせ方──それは「信頼の材料」ではなく「信頼をデザインするスキル」だ。優秀な営業は皆これを持っている。詐欺師も。

　──だから何だ。良い人なんだろう。それで十分だ。

「ねぇ、Ren」

　Pixelが隣に来た。

「塔のてっぺん、行ってきたよ。すっごい景色だった。七つのリージョン全部見えた──あれ、写真見る？」

　Pixelがスクリーンショットを見せてくれた。Nexus Spireの頂上から撮影したパノラマ。北にVerdant Expanseの緑の大地、東にCrimson Peaksの赤い山脈、南にCrystal Shoresの氷の海岸線──世界が丸く見える。地平線が微かに湾曲している。

「……いい写真だな」

「でしょ？　これ、作品集に入れるんだ。僕、美大受験に使うポートフォリオを──」

　Pixelが急に言葉を止めた。

「あれ。……なんだっけ。ポートフォリオを何に使うんだっけ──」

「美大受験だろ」

「あ、そうだそうだ！　ごめん、なんか一瞬、頭が白くなった。──ねぇRen、俺、最近ログインする前のこと、あんまり思い出せないんだよね。施設に来る途中の電車の中で何してたとか、全然出てこない」

「没入しすぎだろ」

　蓮は笑った。

　Pixelも笑った。

　だが──蓮の中のQAエンジニアが、ほんの一瞬だけ、その症状を「バグ報告の書式」で記述しようとして、やめた。

　──ただの没入感だ。仕様範囲内だ。

　広場の向こうで、エミリアが誰かと話している。NLP研究者の彼女は、NCEとの対話テストをずっと続けているらしい。

「ねぇ、Ren。エミリアって知ってる？　あの人、NCEのAIにずっと話しかけてて、なんか名前つけてるんだって」

「名前？」

「うん。『Mila』って。NCEの応答パターンに名前をつけて、研究助手みたいに使ってるらしいよ。面白いよね、AIに名前つけるなんて」

　蓮は遠くのエミリアを見た。静かに端末に向かって何かを入力している。

　──研究者の習慣だろう。実験対象に名前をつける人は多い。ラボのマウスに名前をつける研究者と同じだ。

　蓮はそう思い、それ以上考えなかった。

## 6. 就寝

　夜が来た。

　ArcadiaのVR空間にも昼夜サイクルがある。太陽が沈むと、空に三つの月が昇った。一つは白く大きく、一つは青く小さく、一つはオレンジ色で三日月型だった。幻想的だが、天文学的には明らかにおかしい──蓮は「三連月の公転軌道シミュレーションを確認したい」と思い、すぐにやめた。

　テスターに割り当てられた宿舎は、Central Nexusの居住区にある個室だった。天井が高く、大きな窓から月光が差し込んでいる。ベッドのシーツの触感がリアルだ。繊維の織り目が指先で分かる。

　蓮はベッドに横になり、天井を見た。

　──いい一日だった。

　バグは三件。レポートは提出済み。Pixelと会った。エミリアの名前を聞いた。Advocateの微笑みを見た。

　……兄さんのデータは、まだ探していない。

　明日だ。明日、Undercroftの奥を探索すれば、開発用テストNPCの残骸が見つかるかもしれない。MK2時代のテストデータが残っていれば──兄さんが最後に見た景色が分かるかもしれない。

　蓮はシステムメニューを開き、明日の予定を確認しようとした。

　──その時、視界の端に、あるものが映った。

　メニューの最下段。「ログアウト」の項目。

　そこにある。存在している。だが──灰色だった。テキストも、アイコンも、全体が淡い灰色で描画されている。タップしようと指を近づけると、UIが微かに後退する──まるでこちらの指から逃げるように。

　蓮は何度か試した。指がUIに触れる直前で、ボタンの当たり判定が消える。触れない。押せない。

　──βテスト中はログアウトUIが無効化されている。カプセル側での制御で自動切断されるのだから、VR内にログアウト機能がないのは仕様の範囲内だ。

　蓮はそう結論づけ、メニューを閉じた。

　だが──項目が「ない」のではなく、「ある」のに押せない灰色──その設計を、QAエンジニアの自分が素直に「仕様」と呼べるかどうか。

　──気にしすぎだ。

　蓮はメニューを閉じるとき、UIの通知アイコンの端に、一行のテキストが一瞬だけ表示されたのを見た。

```
session_attribute: PARTICIPANT_ALPHA
```

　──なんだ、これは。

　目を凝らしたが、もう消えている。再表示できない。通知ログにも残っていない。UIの描画バグか、あるいは内部ビルドのデバッグ情報がリークしたか。

　──明日、バグレポートに入れよう。

　蓮は目を閉じた。

　今日は、いい一日だった。

　明日も、楽しみだ。
