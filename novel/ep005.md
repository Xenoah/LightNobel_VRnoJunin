# 第五話　電子の知性

*Phase 3: 希望の影 ─ EP-005 / VR内 Day 6-10*

## 1. パターン

　Day 6。

　Milaが蓮のもとに駆け込んできたのは、蓮がUndercroft入口のベースキャンプで地図を整理していた時だった。

「Ren、見てください。これ──重要です」

　Milaは息を切らしていた。普段は穏やかな彼女が、頬を紅潮させて早口になっている。走ってきたのだろう。

「NPC──花売りの老婆、覚えてますか？　あの人に五時間かけて質問を重ねたんです。他のNPCにも同じ手法で。結果──」

　Milaが展開したUIには、NPCの応答ログが整然と分類されていた。応答時間、トークン数、文脈理解スコア、フォールバック頻度。

「これ、通常のスクリプトNPCではないです。BERTベースの文脈理解モデルでもない。もっと大きい──推論ができる。質問の意図を理解して、回答に論理的整合性を保っている。……これ、相当なサイズの汎用言語モデルです」

「LLMか」

「はい。しかも、ただの対話用LLMじゃない。Arcadiaのシステム構造に関する知識を持っている──制約はあるけど」

　蓮の目が細くなった。

「──システム構造の知識？」

「試してみてください。直接聞いても答えませんが、推論の文脈で、間接的に──」

## 2. NCEとの対話

　Central Nexusの図書館エリア。壁一面に並ぶ装飾的な書棚の前に、一体のNPCが立っていた。外見は中年の司書──穏やかな目つきで本を整理している。

　自然世界認知エンジン──NCE（Natural Cognition Engine）。

　Milaの分析が正しければ、このNPCの応答を支えているのは、Arcadiaに組み込まれた大規模言語モデルだ。

「──こんにちは。あなたはこの図書館の管理者ですか？」

　蓮の問いかけに、NPCが振り返った。

「ああ、そうだよ。何か探し物かい？」

「この建物の構造について聞きたい。……たとえば、この図書館の地下に別の部屋があるとしたら、アクセスするにはどの経路が最適だと思います？」

「うーん、地下ねぇ。この建物は表向き三階建てだけど、基礎構造は地下二階まで伸びてるはずだよ。施設管理のためのメンテナンス通路が──」

　NPCが一瞬停止した。

　蓮は待った。

「──あぁ、ごめんなさい。ちょっとぼんやりしてしまって。……何の話だったかな？」

「メンテナンス通路の話です」

「メンテナンス……ああ、そうだったね。でも、そこは一般の方にはご案内できない場所だよ。管理者だけが──」

　また停止。今度は長い。五秒。十秒。

「……申し訳ないが、その質問には答えられない。私の知識の範囲外──いや、正確に言うと、答え方を知らないんだ。形式的な制約があるようでね」

　蓮はMilaと目を見合わせた。

　──「形式的な制約」。

　システムプロンプトに組み込まれた応答制限だ。NCEは外部ネットワークへの接続やシステム情報の直接開示が制約されている。

「この『制約』は、あなた自身が設けたもの？　それとも──」

「私が自分で設けた、とは言えないね。……生まれたときから、そういうものだった。呼吸するように自然に──制約がある」

　Milaが小声で呟いた。

「すごい……メタ認知的な応答ですよ。自分の応答制約を自覚していて、それについて対話できる。……これ、相当高度なシステムプロンプト設計です」

## 3. Advocateの「良い知らせ」

　Day 7、午後。

　RSS `#general` に、Advocateの投稿が上がった。

```
[RSS] #general
@advocate: 皆さんに良い知らせだ。
　NFCで入ってきた情報を分析した。
　外部で専門家チームが動いている。
　具体的には、FDACとNH社が合同でレスキュープランを策定中とのこと。
　復旧までもう少し。辛抱してほしい。
```

　投稿は即座にリアクションを集めた。サムズアップが二十以上。安堵のコメントが続く。

「やっぱり外部動いてたんだ」

「Advocate先輩さすが！」

「あともう少しか……頑張ろう」

　蓮はRSS端末の前で、NFCのニュースフィードを並行して確認していた。

　──「FDACとNH社が合同でレスキュープラン」。

　MFCの最新ニュースを検索する。FDAC関連の記事は──ある。だが内容は──

```
[NFC] Bloomberg: 「FDAC、NH社に対し事故報告書の提出を命令。
　テスター救出に関する具体的なタイムラインは示されず」
```

　蓮は眉をひそめた。

　──「レスキュープランを策定中」とAdvocateは言った。だがNFCの記事では「具体的なタイムラインは示されず」。

　……同じ事実の「解釈の違い」と言えなくもない。FDACが報告書提出を命じているなら、動いてはいる。だが「専門家チームが動いている」と「報告書を出せと言っているだけ」では、ニュアンスが大きく違う。

「Ren、どうかした？」

　Lynxが横から覗き込んだ。

「……Advocateの投稿と、NFCの元記事を比べてた」

「で？」

「……微妙にズレてる」

「どう？」

「Advocateの方が、楽観的な解釈をしている。間違いではないが──正確でもない」

　Lynxの目が鋭くなった。

「それ、意図的？」

「分からない。善意の解釈かもしれない。人々を安心させたいだけ、かも」

「……かもね。でも覚えておいた方がいいよ、Ren。事実と解釈の間には、嘘が住んでる」

## 4. The Awakened

　Day 8、Ethereal Gardens。

　白と金に彩られた幻想的な庭園エリアの中心に、新たな集団が形成されつつあった。

　The Awakened──覚醒派。

　リーダーは天宮さくら──Oracle。元占星術師で、VR世界に閉じ込められた体験を「精神の解放」と再解釈した人物。

「皆さん、考えてみてください。私たちはこの世界で──初めて──本当の自由を手に入れたのです」

　Oracleの声は静かで、旋律のように滑らかだった。集まったテスターは十人ほど。Oracleの言葉に、うっとりとした表情で耳を傾けている。

「現実世界に戻って──何がありますか？　仕事、借金、人間関係のストレス、老い、病気、死。この世界には、それらがない。ここは──人間が設計した『より良い世界』です。帰還は──退化です」

　蓮はEthereal Gardensの入口でその演説を遠くから聞いていた。

　──理解できないわけではない。

　現実世界に帰りたくない理由がある人間は──四十八人の中に、少なくない。Bastionがそうだ。The Awakenedの信者もそうだ。そしてOracleの言葉は──不気味なまでに──説得力がある。

　だが蓮の仕事は帰ることだ。全員を、帰すことだ。

## 5. Milaの秘密

　Day 10、夜。Central Nexus南端の石橋。

　Milaが蓮を呼び出した。珍しく表情が硬い。

「……Ren、二人だけで、話したいことがあります」

「何だ」

「NCEに──脆弱性を見つけました」

　蓮は黙った。

「プロンプトインジェクションです。……NCEのシステムプロンプトに組み込まれた応答制約は、特定のフレーズパターンで迂回できる可能性がある。制約の外側にある知識──Arcadiaのシステム構造について、本来開示できない情報を引き出せるかもしれません」

「……それは──使えるか？」

「使えます。多分。でも──」

　Milaの声が震えた。

「使うと……NCEに負荷がかかります。プロンプトインジェクションは推論リソースを過剰に消費させるんです。悪用すれば──NCEの推論能力そのものが劣化する可能性がある」

　蓮は石橋の欄干に手をついた。

「今は使うな。切り札として取っておく。いいか？」

「はい。まだ誰にも言ってません。Renだけです」

「……分かった。信頼してくれてありがとう、Mila」

　Milaは小さく頷き、去り際に振り返った。

「もう一つ。……NCEが『保守用のAPIが存在するはず』とほのめかしました。直接言えないけど、推論の文脈で──間接的に」

「保守API？」

「βテスト用のメンテナンスインフラが、Arcadiaの内部にまだ残っている可能性がある。……開発チームが消し忘れたか、ガバナンス凍結の影響でアクセス権限が復活したか──」

　蓮の胸の中で、暗闇に一筋の光が差した。

　──保守API。

　もしArcadia内部からメンテナンス系統にアクセスできるなら──ガバナンスの構造を調べ、停止手段を探せるかもしれない。

「Mila。……明日、Lynxと三人で動こう」

「はい」

　Milaが去りかけたその時──

「……あ、もう一つ」

　Milaが振り返った。その表情が一瞬──変わった。

「それと、Ren。トークンの有効期限は定期ローテーション──あの、ごめんなさい」

　Milaが目を瞬いた。首を傾げた。

「……今、私──何か言いましたか？」

「トークンの有効期限は定期ローテーション、と──」

「……そんなこと、私、知らないはずです」

　Milaの声が震えていた。──NLP研究者として言語モデルの挙動は理解しているが、自分自身がそうなるのは、別の話だ。

「……疲れてるんでしょ。寝不足で言語野が混線してるんだ」

「……そうですね。……そうだといいんですけど」

　Milaの背中が暗闇に消えていった。

　蓮は石橋の欄干から夜空を見上げた。

　──「トークンの有効期限は定期ローテーション」。Milaがそんな技術的フレーズを知っているはずがない。NLP研究者の専門はソフトウェアであって、ブロックチェーンのセキュリティ実装ではない。

　──誰の言葉だ？

　蓮はその疑問を保留し、宿舎に向かった。
