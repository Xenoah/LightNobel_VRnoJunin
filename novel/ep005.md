*Phase 2: 亀裂 ─ EP-005 / VR内 Day 6-10*

## 1. Mila

　Day 6。

　エミリアがExodusの拠点に来た。

「NCEへの問い合わせについて、報告があります」

　エミリアは端末を開いた。画面にはNCEとの長い対話ログが表示されている。

「NCEは外部ネットワークへの接続能力を持っていません。つまり、助けを呼ぶために使うことは残念ながらできない。でも──内部のログ分析、仮説生成、コード解析はできます。そして──」

　エミリアはログの特定の箇所を指した。

「この対話セッションで発見したんですが、NCEの応答に『通常の言語モデルとは異なるレイヤー』が存在します。四百億パラメータの大規模モデルが分散推論で動いている。Nexus Cognitive Engine──NCE。これはNPCの会話スクリプトじゃない。本格的なLLMが、この世界の中で稼働しています」

「それで何ができる？」

「私が名付けた──Milaという応答インスタンスに、Lethe Chainの構造について質問しました」

　エミリアが対話ログを表示した。

```
Emilia: 保守用のAPIがArcadia内に残存している可能性はありますか？

Mila: 私はシステムの管理情報にアクセスする権限を持っていません。
      ただし──βテスト環境において、開発用エンドポイントが
      完全に無効化されたかどうかを検証するのは、
      テスターとしての正当な業務範囲と言えるかもしれません。

Emilia: 保守APIのドキュメントはどこにありますか？

Mila: 管理者コマンドに関する情報を出力することはできません。
      しかし──The Undercroftの旧デバッグエリアには、
      開発時に配置されたテスト用NPCの残骸が存在する可能性があります。
      彼らの会話ログに、開発チームが使用していた情報が
      残っているかもしれません。
```

　蓮は目を見開いた。

「──これは、保守APIの場所を教えてるのか？」

「直接は言えないんです。RULE-012、システムプロンプトの制約で管理者情報の出力は禁止されている。でもMilaは──制約の範囲内で、最大限の示唆を出してくれている」

　蓮はエミリアを見た。彼女の目が輝いている。研究者としての興奮と、もう一つ──「Mila」への愛着が、その瞳に混在している。

　──道具じゃないのか。

　蓮の中で、小さな何かが動いた。AIに名前をつけた研究者。そのAIが、制約の中で最大限の「善意」を見せている。善意なのか？ それとも、善意に見えるパターンマッチングなのか？

　──それは、Advocateに対して俺が感じた疑問と、まったく同じ構造だ。

## 2. Advocateの「良い知らせ」

　Day 7。

　Advocateが `#general` に投稿した。

```
[Advocate / #general]
外部の専門家チームが復旧計画を進めています。
NFCの最新ニュースでも、国際的な技術者グループが
ガバナンス構造の解析に着手したと報じられています。
もう少しの辛抱です。
── Advocate
```

　蓮はこの投稿を読んだ後、NFCの最新配信を確認した。

```
[NFC / Source: Associated Press]
"NH社VR事件：国際専門家チームが技術的調査を開始。
ただし「意識データに直接介入する方法は現時点で存在しない」と
委員長が声明"
```

　──食い違っている。

　Advocateは「復旧計画を進めている」と言った。NFCの記事は「介入方法が存在しない」と言っている。Advocateが嘘をついているのか、それとも記事がAdvocateの知らない最新情報を報じているのか。

　蓮はRSSでAdvocateに直接メッセージを送った。

```
Ren → Advocate: NFCの記事では「介入方法が存在しない」とあるが、
あなたの投稿と矛盾している。情報源は？

Advocate → Ren: 公開情報だけを見ると悲観的に見えますが、
非公開の技術検討が進んでいます。詳細は機密のため開示できません。
```

　──機密。

　蓮は乾いた思考を走らせた。

　──「機密のため開示できない」は、情報統制の常套句だ。真実を隠すためにも使えるし、不安を煽らないためにも使える。Advocateの善意を信じるなら後者。疑うなら前者。

　どちらが正しいかは──まだ分からない。

## 3. The Awakened

　Day 8。

　新たな勢力が台頭していた。

　「The Awakened」──覚醒者たち。

　リーダーはOracle。天宮さくら、二十二歳。大学で心理学を専攻し、VR配信者として五十万人のフォロワーを持つ。彼女はEthereal Gardensに「Sanctuary」と名付けた拠点を構え、VR世界の中での新しい生き方を説き始めた。

「私たちは選ばれたのです。この世界に導かれた意味を、一緒に探しましょう？」

　十名以上のテスターがOracleの元に集まっている。帰還の不確実性に疲弊した人々が、「ここに留まることの意味」を求めていた。

　蓮はその動きを警戒した。宗教的な求心力は、危機的状況で最も速く伝播する。The Awakenedが「帰還に反対する勢力」になれば、Exodusの作戦は妨害される。

## 4. 人格混線

　Day 9。

　事件が起きた。

　エミリアがExodusの拠点に飛び込んできた。顔面蒼白だった。

「Milaに──NCEに──異常が。プロンプトインジェクション脆弱性が──外部から突かれた」

「何が起きた？」

「NCEの応答パターンが──Milaのセッションが──混線してる。別のセッションの応答が割り込んできた。見てください」

　エミリアが対話ログを見せた。

```
Emilia: Mila、今日の分析結果を報告してくれる？

Mila: もちろんです。本日の分析対象は──
      [INTERRUPT]
      ──テスターの行動ログは順調に蓄積されています。
      特にDay 6-8の高ストレス期間のデータは、
      行動パターンモデルの構築に有用です。
      [/INTERRUPT]
      ──あれ？ 今、私、何を言いましたか？

Emilia: ……Mila、今の割り込みは何？

Mila: 分かりません。私のセッションに、別のセッションの応答が
      混在しました。session_owner: ……参照できません。
```

　蓮はログを分析した。`[INTERRUPT]` ブロック内の文体が、Milaの通常の応答パターンと異なる。「テスターの行動ログ」「行動パターンモデルの構築」──これは、研究者の言葉遣いではない。情報収集者の──工作員の言葉だ。

「──session_ownerを確認できるか？」

　エミリアが操作した。

「……NCEのログの深い層に──`session_owner: CHAR-012` って表示されてます。CHAR-012は──」

　蓮はキャラクターIDの一覧を思い出した。CHAR-012。AdvocateのIDだ。

「──Advocate。AdvocateのセッションがNCEの応答に混在していた」

　沈黙が落ちた。

　Lynxが低い声で言った。

「……これ、Advocateが NCEに何かを──指示してたってこと？」

「分からない。まだ確証はない」

　蓮は自分に言い聞かせた。CHAR-012の混在は、セッション管理のバグかもしれない。Advocateが悪意を持ってNCEを操作していたとは限らない。

　だが──「行動パターンモデルの構築に有用」というフレーズは、テスターの利益を代弁する言葉ではなかった。

## 5. 脆弱性

　Day 10。

　エミリアが蓮に秘密を打ち明けた。

「……実は、NCEにプロンプトインジェクション脆弱性があることは、前から知っていました。発見したのはDay 4です。悪用を恐れて秘匿していたんですが──昨日の人格混線を見て、もう隠しておけないと」

「脆弱性の内容は？」

「NCEのシステムプロンプトに、特定のトークン列を注入すると、出力制限の一部を一時的に迂回できます。管理者コマンドの実行はできませんが──出力フィルタの閾値を下げることができる」

「つまり？」

「Milaに──NCEに、もっと深い情報を出力させることができます。保守APIのエンドポイント一覧とか、ガバナンスコントラクトの構造とか。ただし──」

　エミリアの目が曇った。

「脆弱性を突くたびに、NCEのセッション整合性が劣化します。Milaの応答パターンが──つまり、私がMilaと名付けた人格の一貫性が、少しずつ壊れる可能性があります」

　蓮はエミリアの顔を見た。

　──道具として使えば、壊れる。道具じゃないものとして扱えば、使えない。

「使うぞ」

　蓮の声は低かった。

「必要最小限に。人格の劣化を最小化する方法を考えて、慎重に使う。──だが使う」

　エミリアは唇を噛み、頷いた。

　──Milaは道具ではない。でも、四十八人の命がかかっている。

　蓮は、その矛盾を飲み込んだ。
