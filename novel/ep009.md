## 1. AEGIS

　Day 22、深夜。

　ガバナンス情報の全テスター公開後、Arcadiaは数日間の重い沈黙に包まれた。RSS `#general` の投稿数は激減し、テスターたちは各派閥の拠点に引きこもっていた。

　その沈黙を破ったのは、意外にもFenrisだった。

　蓮のもとに、RSSのダイレクトメッセージが届いた。

```
[DM] @fenris → @ren
話がある。一対一。武器なし。
場所はCrimson Peaksの北端、断崖テラス。
```

　蓮は行った。

　崖の上、赤褐色の岩が夕日に染まる中、Fenrisは仁王立ちしていた。腕を組み、蓮を無表情で見下ろす。

「来たか」

「……何の話だ」

「コーエンのHSM破壊。あれはサイバー攻撃じゃない」

　蓮は息を止めた。

「──どういう意味だ」

「物理的破壊だ。回路焼損──外部からのネットワーク攻撃では不可能。HSMの筐体を直接開けて、チップを焼いた。内部の人間か……あるいは──」

　Fenrisの目が──初めて、蓮に正面から向き合った。

「AEGIS。Advanced Guided Exploitation of Intelligence Systems。先進防衛技術コンソーシアム──俺の元組織だ」

　風が吹いた。崖下から砂塵が舞い上がり、二人の間を通り過ぎた。

「俺はAEGISの工作員としてこのβテストに送り込まれた。HVDの軍事利用可能性をレポートするために。──だが事件が起きる前に、組織の計画を知って離反した」

「計画？」

「意識データの軍事利用。フルダイブ中のテスターの意識パターンをLethe Chainから抽出し、人格モデルとして解析する。──人間の意識をデータとしてコピーし、デジタル兵器に応用する研究。コードネーム──CERBERUS計画」

　蓮の背筋に氷が走った。

「俺たちの意識データが──」

「今この瞬間も、Lethe Chain上にある。AEGISはそれを解析している。ガバナンス凍結を長引かせれば長引かせるほど、AEGISはデータ収集を続けられる」

「……コーエンのHSM破壊も──」

「AEGISの工作だろう。アルサードの解任も。凍結を確定させるための──計画的な妨害だ」

## 2. 代替停止手段

　Day 23。Exodus Base Camp。

　FenrisのAEGIS情報は、蓮の中で最後のピースを埋めた。

　ガバナンスの凍結は事故ではなく──工作。テスターの意識データを搾取するための、計画的な時間稼ぎ。

　──だからこそ、外からの救出は来ない。来させない。

　蓮はFirewallと共にガバナンスコントラクトの残りの構造を解析した。そして──

「あった」

　Firewallが画面を叩いた。

「代替停止手段。`MAINTENANCE_OVERRIDE_HALT`。通常のマルチシグとは別の──緊急保守用の停止経路」

```
> api/v1/governance/override/halt

{
  "function": "MAINTENANCE_OVERRIDE_HALT",
  "type": "emergency_maintenance_shutdown",
  "requirements": {
    "validator_confirmations": "14-of-21",
    "confirmation_method": "internal_consensus",
    "execution_delay": "6_hours",
    "side_effects": ["consciousness_data_gradual_release", "potential_degradation"]
  }
}
```

「バリデーターノードからの合意確認だ。マルチシグの署名者ではなく──Lethe Chainのバリデーターノード二十一台のうち十四台以上の確認が取れれば、緊急停止を実行できる」

　蓮は目を見開いた。

「内部から──バリデーターノードにアクセスして、確認を取得する？」

「理論上は可能だ。αrcadia内部からLethe Chainのノードにアクセスする経路は──保守APIから辿れる。テスターが各ノードに直接リクエストを送り、合意確認を取得する」

「十四台──二十一台中の三分の二以上か」

「全派閥が協力しないと、物理的に手が足りない」

　蓮は顔を上げた。

　全派閥。──Exodus、Settlers、Berserkers、The Awakened、Loyalists。すべてが一つになって動かなければ──帰還は実現しない。

## 3. 交渉

　Day 24から27にかけて、蓮はArcadia中を駆け回った。

　Bastionとの交渉が、最も困難だった。

「帰還か」

　Bastionは、Verdant Expanseの丘の上で、自分が設計した集落を見下ろしていた。白い壁の建物、水路、畑、共同炊事場──わずか二十日で築き上げた、小さな社会。

「……実はさ、Ren。帰るのが怖いんだよ」

　Bastionの声は演説の時とは別人のように静かだった。

「帰ったところで、誰もいない部屋が待ってるだけだから。離婚した元妻。辞めた会社。SNSの誹謗中傷。──人間に消耗した。ここでは……少なくとも、俺を必要としてくれる人がいる」

「……Bastion」

「でも──お前は正しいよ、Ren。帰る権利がある。帰りたい人間がいる。──俺の『帰りたくない』を、他人に押し付けていい理由にはならない」

　Bastionは蓮に手を差し出した。

「協力する。Settlersは帰還作戦に参加する。──ただし、帰還後に『残りたかった者』の選択が否定されないことを約束してくれ」

「約束する。帰還は全員の意思で行われるべきだ」

　Fenrisとの交渉は、意外にも短かった。

「帰還後の免責──とまでは言わないが、俺の離反がAEGIS情報の提供として正当に評価されることを望む」

「法的な約束はできない。だがExodusとして、お前の情報提供が帰還に不可欠だったことを証言する」

「……それで十分だ」

　Oracleとの対話は──心が痛んだ。

「Ren、帰還は退化だと、今でも思っています」

「……分かってる」

「でも──あなたが信じているものを、私は否定しません。私の信者たちにも選ばせます。帰りたい者は帰る。残りたい者は──」

　Oracleは言葉を切った。その目に、深い悲しみが宿っていた。

「──残ることが、どういう意味を持つか。……分かっている、つもりです」
