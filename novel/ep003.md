# 第三話　七十二時間の嘘

*Phase 2: 亀裂 ─ EP-003 / VR内 Day 2-3*

---

## 1. 七十二時間

　Day 2、午後八時。

　七十二時間のテスト期間まで、残り十六時間。

　蓮はUndercroftの第二層で、兄のデータを探していた。MK2時代のテスト用NPCの残骸──開発時に配置され、本番デプロイ前に削除されるはずだったテスト用エンティティ。そのメモリ領域に、兄が最後に対話した相手のログが残っている可能性がある。

　だが見つからない。

　テスト用NPCの識別子を手がかりにメモリアドレスを辿っても、該当する領域はすべて初期化済みだった。MK3のβ版デプロイ時にクリーンインストールされたのだろう。データは上書きされた。

　──分かっていた。分かっていて、来たんだ。

　蓮は洞窟の壁に背を預け、天井を見上げた。鍾乳石が松明の光を受けて輝いている。

　──兄さん。

　ここにはもう何も残っていない。でも、この世界のどこかに──Lethe Chainの深い層に──MK2時代のスナップショットの断片が眠っている可能性はまだある。ただし、それを探すにはガバナンスコントラクトの読み取り権限が必要で、テスターロールでは届かない。

　明日、帰ったら、NH社に正式にデータ開示を請求しよう。

　蓮はそう決めて、Undercroftを後にした。

## 2. NFC

　Day 3、午前六時。

　起きてRSSを開いた瞬間、蓮は凍りついた。

　`#news` チャンネルにNFC──News Feed Channel──が大量の記事を配信していた。NFCはArcadia内部のニュースアグリゲーターで、外部のニュースサイトをクロールしてVR内に配信する仕組みだ。通常は一日に数件程度の配信だが──今朝は、四十件以上が一気に流れ込んでいる。

```
[NFC-ALERT / PRIORITY: CRITICAL]
Source: Reuters
"NH社βテスト事故：48名が30日経過後も覚醒せず"

[NFC-ALERT / PRIORITY: CRITICAL]
Source: NHK
「人間仮想化装置テスト中の事故──48名の意識がVR空間に閉じ込められた可能性」

[NFC-ALERT / PRIORITY: CRITICAL]
Source: BBC
"Nexus Horizons VR test: 48 participants trapped in virtual reality for 30 days"
```

　三十日。

　蓮は記事を読み直した。七十二時間ではない。三十日。テスト期間は七十二時間のはずだった──だが外部ニュースは、事故発生から三十日が経過していると報じている。

　──何かの間違いだ。

　だが数字は一致する。ロイターもNHKもBBCも、すべてが「三十日」と報じている。現実世界では二〇三一年八月十七日。テスト開始日は七月十八日。差分は三十日。

　蓮は自分のシステムログを開いた。

```
login_timestamp: 2031-07-18T09:00:00Z
current_session_duration: 720:14:33
```

　七百二十時間。三十日間。

　──俺は、三十日間ここにいた。

　体感では──二日間だった。Day 0にログインし、Day 1にLynxと会い、Day 2にPixelと塔に登った。三日間の記憶しかない。だが実際には三十日が経過している。

　意識のタイムスタンプと現実のタイムスタンプが乖離している。QAエンジニアの思考が加速する──これはバグだ。意識同期モジュール（NCB）のδ同期が、主観時間を圧縮しているか、あるいは記憶の書き込みが断続的にスキップされている。

　RSSが爆発した。

```
Pixel: え、え、え？ 30日ってどういうこと？？？
Bastion: 落ち着け。まずソースの信頼性を確認しろ
Lynx: NFCのクローラは外部ニュースサイトをリアルタイムで取得してる。ソースは本物
Forge: ログアウトできないのは仕様じゃなくてバグだったってことか？？
Gazette: 全員、パニックにならないでください。情報を整理します
```

## 3. 混乱

　Central Nexusのハブは群衆で溢れていた。

　四十八人全員がハブに集まっている。叫んでいる者、泣いている者、壁を殴っている者。Pixelは顔面蒼白で座り込んでいた。Bastionが怒鳴っている。Lynxだけが端末に向かって冷静にログを解析している。

「みなさん、落ち着いてください」

　Advocateの声が、群衆の上に響いた。

　彼はハブの中央に立ち、両手を広げていた。表情は穏やかだが、声には力がある。

「ニュースの内容は事実です。我々は三十日間、Arcadiaの中にいます。意識同期モジュールのバグにより、我々の意識はLethe Chainに格納された状態になっています」

　沈黙が落ちた。

「しかし──」

　Advocateは言葉を区切った。

「しかし、外部のNH社は全力で解決に動いています。ガバナンスコントラクトの署名者七名が、緊急停止コマンドの発動に向けて調整中です。七十二時間以内に──今度こそ本当の七十二時間以内に──解決の目処が立つ見込みです」

「根拠は？」

　蓮が聞いた。

　Advocateは蓮を見た。

「NH社のCTOであるエレナ・ヴァイスが、NFCを通じて内部向けメッセージを送信しています。確認しますか？」

　RSSの `#official` チャンネルに、確かにNH社名義の通知があった。

```
[NH-INTERNAL / FOR TESTERS]
From: Elena Weiss, CTO
Subject: 現状報告と復旧計画

テスターの皆さまへ。

ご心配をおかけしています。現在、ガバナンスコントラクトの
緊急停止コマンド発動に向けて、署名者7名との調整を
最優先で進めています。

安全な覚醒シーケンスの実行には5名の署名が必要です。
現時点で3名の署名が確保されており、残り2名との
交渉が進行中です。

カプセルの生命維持は正常に動作しています。
皆さまの身体の安全は確保されています。

72時間以内に次の進展をお知らせします。

── Elena Weiss
```

　三名確保。残り二名。五名で足りる。

　蓮はその数字を見て、少しだけ安堵した。──だが同時に、QAエンジニアの思考が別の計算を走らせている。

　──七名中三名が確保。四名が未確保。未確保の理由が何であれ、三十日間で三名しか集まっていないのは遅い。署名は「やるかやらないか」のバイナリ判定だ。三十日かけて四名が署名していない理由は、「調整中」ではなく「不可能」である可能性がある。

　だが──蓮はその思考を声にしなかった。

　今この場で必要なのは、パニックの鎮静だ。

## 4. 【現実カットイン①】

　──同時刻。現実世界。NH社モニタリングルーム。

　エレナ・ヴァイスは、四十八枚のモニターを見ていた。

　各モニターに、一人のテスターのバイタルデータが表示されている。心拍数、血中酸素濃度、脳波パターン、カプセル内液温。すべて正常範囲内だ。

「エレナ」

　背後から声がかかった。クロスだ。ヴァルター・クロス。NH社CEO。四十八歳。三十日前には「ビジョナリー」と呼ばれていた男が、今は法律顧問三人に囲まれて立っている。

「署名の状況は」

「変わりません。三名が確保。コーエンの鍵は物理的に喪失。ウィンストンは安全性証明を要求。アルサードは取締役会に止められています。あなたの鍵はFDACが差し押さえたまま」

「弁護士に裁判所命令を──」

「出しています。ですが連邦レベルの手続きに数週間かかります」

　エレナはモニターに向き直った。

　四十八人の心拍数は安定している。彼らは生きている。彼らの意識はLethe Chain上で「実行中」であり、VR世界の中で生活している。

　──助けたい。

　しかし助ける方法が、制度の中に閉じ込められている。

　エレナは通知ログを確認した。テスター向けメッセージの送信確認。`ENTITY_DELTA_SYNC_OK ×48`。四十八人のδ同期が正常。

　──だが一瞬、UIに別の行が表示された。

```
ENTITY_DELTA_SYNC_OK ×49
```

　四十九。

　エレナは目を凝らした。だがその行はもう消えていた。画面には`×48`の行だけが残っている。

　──ログの表示バグか。

　エレナはそう判断し、次の対応に移った。

## 5. Advocateの安心

　VR内。

　Advocateの説明は落ち着いていた。データを示し、時系列を整理し、外部の動きを解説した。パニックは徐々に収まり、テスターたちは各自の宿舎に戻っていった。

　蓮はハブに残っていた。Lynxも残っている。エミリアも。

「Ren。あたし、ガバナンスコントラクトの構造を前から研究してた。NDAで論文には出せなかったけど──構造は知ってる」

「分かってるのか。五名の署名が集まる見込みは」

　Lynxの表情が曇った。

「……正直に言うと、厳しい。コーエンの鍵が物理的に消えてるなら、五名は数学的に困難だ。四名が使えないなら、残りの三名で五名分は──」

「足りない」

「足りない」

　沈黙が落ちた。

　エミリアが静かに言った。

「Milaに──NCEに聞いてみました。ガバナンスコントラクトに代替の停止経路があるかどうか」

「何か分かったか？」

「NCEの回答は制限されています。管理者コマンドに関する情報は出力できません。でも──『保守用のエンドポイントが残存している可能性は否定できない』という回答がありました。否定できない、という言い方が気になりました」

　蓮は考えた。

　──否定できない。NCEは嘘をつかない（はずだ）。だが肯定もしない。制約の中で、最大限の情報を出している。

「エミリア。Milaに──NCEに、もう少し聞けるか？」

「……できますけど、聞くたびに監査ログが増えるんです。NCEへの問い合わせ頻度が高いと、システム側の注意を引く可能性があります」

　──使うほど地獄。

　蓮はその制約を理解した。便利な道具に見えるほど、使った記録が残る。

「必要最小限にする。まずは保守APIのエンドポイントの存在確認だけだ」

　エミリアは頷いた。

　──帰れる。必ず帰れる。

　蓮はそう自分に言い聞かせた。だが心の中のQAエンジニアは、別の仮説を立て始めていた。

## 6. 0 bytes

　夜。

　蓮は宿舎に戻り、今日のバグレポートを整理していた。

　──ふと、NFCの記事一覧を見直した。

　エレナ・ヴァイスの内部向けメッセージ。その添付ファイルに目が止まった。「事故対策委員会 議事録」というPDFファイル。蓮はそれを開こうとした。

```
Error: File size is 0 bytes.
Cannot open document.
```

　0バイト。

　ファイル名は存在する。日付のタイムスタンプもある。だが中身が空だ。

　蓮はNFCのメタデータを確認した。ファイルはNH社の公式サーバーから配信されている。ファイル名：`incident_committee_minutes_20310817.pdf`。ファイルサイズ：0。

　──NFCのクローラが取得時に中身をドロップしたか？　それともアップロード元が空のファイルを送信したか？

　どちらにしても、事故対策委員会の議事録が「存在するが中身がない」という状態は──仕様とは呼べない。

　蓮はバグレポートに記録した。だが優先度は「低」にした。議事録の中身がなくても、エレナのメッセージで状況は把握できている。コーエンの鍵が壊れていて、他の署名者にも問題がある。だが三人は確保されている。

　──帰れる。帰る方法は、まだ見つかっていないだけだ。

　蓮は端末を閉じ、目を閉じた。

　VR内の夜は静かだ。窓の外に三連月が浮かんでいる。月光が部屋に差し込み、壁に影を作っている。

　──この影の形は、昨夜と同じだろうか。

　蓮はその思考を追うのをやめた。

　眠れない夜が始まった。
