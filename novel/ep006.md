# 第六話　地下回廊

*Phase 3: 探索 ─ EP-006 / VR内 Day 10-15*

---

## 1. 探索

　Day 10。

　Exodusは五人になった。蓮、Lynx、Firewall、Forge（鍛冶師）、Specter（元ハッカー）。Undercroftの第三層へ潜る日が来た。

「ルートは僕が確認した」

　Firewallが壁面に投影したマップを指した。

「第二層までは以前の探索で通過済み。第三層に降りるには、Sector C-12の垂直シャフトを使う。問題は──第三層に徘徊する敵性NPCのレベルが第二層の三倍以上だということ」

「戦闘回避ルートは？」

「二つ。だけど、どっちも一度はモンスターの巡回経路と交差する。最低一回は戦闘が必要」

　蓮は装備を確認した。VR内の戦闘は体力を──正確には、精神力を消耗する。痛覚は九十五パーセント減衰されるが、恐怖と疲労は減衰されない。

「──行こう」

---

## 2. 第三層

　Undercroftの第三層は、上層とはまったく異なる空間だった。

　洞窟ではない。人工的な──いや、「開発的な」空間だ。壁面がグリッドパターンで構成されている。テクスチャが適用されていない、ワイヤーフレームの壁。床にはUV展開用の座標線が走っている。天井にはデバッグ用のライトソースが点灯していて、影が一切存在しない。

「──開発環境だ」

　Lynxが呟いた。

「β版のデプロイで上書きされなかった、レベルデザインの制作途中のフロア。テクスチャが適用されてないのは、本番には含まれない想定だったから」

　Forgeがワイヤーフレームの壁を叩いた。コンッと軽い音がする。物理判定は生きている。

「ここに、テスト用NPCの残骸があるはず」

　蓮はグリッドの通路を進んだ。フロアの中央に、三体のNPCが「配置されたまま」だった。停止状態。ポーズが不自然に固まっている。テスト用に配置され、βデプロイの対象外として放置されたエンティティ。

　蓮はNPCの一体に話しかけた。

```
Test NPC (ID: DEV-NPC-047): ……
Status: ACTIVE (idle loop)
Last interaction: 2031-06-20T14:33:12Z
```

　最終アクセスが七月十八日の事故より前──六月二十日。開発チームがテストしていた時の最後の対話だ。

「対話ログを見せてくれ」

```
[Dialog Log / DEV-NPC-047]
...
Dev_User_09: /api/v1/health の応答を確認してくれ
DEV-NPC-047: 確認しました。応答正常です。
    URL: https://arcadia-internal.nh.local/api/v1/health
    Status: 200 OK
    Response: {"status":"ok","version":"2.0.4","modules":["ncb","governance","maintenance"]}

Dev_User_09: maintenance module のドキュメントは？
DEV-NPC-047: ドキュメントパス: /docs/maintenance/api_reference.md
...
```

　蓮の手が震えた。

「──見つけた」

## 3. ヘルスチェック

　蓮はURLをFirewallに渡した。Firewallが端末から直接、保守APIのヘルスチェックエンドポイントを叩いた。

```
GET /api/v1/health

Response:
{
  "status": "ok",
  "version": "2.1.0-rc3",
  "uptime": "2592000s",
  "modules": [
    "ncb",
    "governance",
    "maintenance",
    "audit",
    "debug"
  ]
}
```

　全員が息を呑んだ。

「──生きてる」

　Lynxの声が震えている。

「保守モジュールが──maintenance が──まだ動いてる！」

　蓮はレスポンスを読み解いた。バージョンは`2.1.0-rc3`──事故を引き起こしたNBCモジュールと同じバージョン。稼働時間は二百五十九万二千秒──三十日間。事故発生時から一度も再起動されていない。

　そして──`modules` に `maintenance` と `debug` がある。

「β版のデプロイで無効化されるはずのモジュールが残存している。──NT社の品質管理の甘さが、逆に俺たちの命綱になった」

　Firewallが冷静に確認した。

「ただし、この先のエンドポイントには認証が必要なはず。テスターロールでアクセスできる範囲は限られる」

「──まずは何ができるか確認する。段階的に攻める」

　蓮はメモを取り始めた。保守API。段階1：ヘルスチェック到達。完了。段階2以降は──監査ログ、デバッグコンソール、権限昇格。一歩ずつだ。

## 4. Base Camp

　Undercroftの第三層に、Exodusの前線拠点──Base Campを設営した。

　ワイヤーフレームの壁に囲まれた小さな空間。テクスチャのない床にForgeが簡易的な作業台を設置し、Specterが通信用のRSS中継器を組み立てた。

「ここからCentral NexusまでのRSS中継は確保した」

　Specterが報告した。

「ただし──ここでの活動はLethe Chainのトランザクションログに記録される。長時間留まれば、外部から見れば異常な活動パターンとして検出される可能性がある」

「検出されるとどうなる？」

「分からない。だけど──Advocateは見てるだろうね」

　蓮は頷いた。Advocateの目を気にしながら作業する必要がある。

　帰路につこうとした時──蓮は足を止めた。

「──止まれ」

　蓮は耳を澄ませた。足音が聞こえる。壁面のグリッド通路に反響する足音。蓮、Lynx、Firewall、Forge、Specter。五人。足音を数えた。

　一、二、三、四、五──六。

　六人分の足音が鳴っている。

「……Lynx。今、何人いる」

「五人でしょ？ あたしとRenとFirewallとForgeと──」

「足音を数えてくれ」

　全員が立ち止まった。静寂になった。足音は消えた。

「……気のせいか」

「かもね。反響が重複したんじゃない？ ワイヤーフレームの壁は反射特性がおかしいから」

　蓮はそう結論づけた。だがFirewallは端末を操作し、帰路のトランザクションログを確認した。

「……`participants: 6/5`」

「何？」

「帰路のトランザクションに、参加者が六人と記録されてる。でもメンバーリストには五人しかいない」

　沈黙。

「──ログのバグだ」

　蓮は言った。

　Firewallは何も言わず、端末を閉じた。

## 5. Lynxの値段

　Base Campを出て、Central Nexusに戻る途中、Lynxが蓮の隣に並んだ。

「ねぇ、Ren。あたし、なんでこのβテストに参加したか知ってる？」

「研究だろ」

「半分正解。もう半分は──金」

　Lynxは空を見上げた。Undercroftの暗い天井が、Central Nexusに近づくにつれて開けていく。

「博士課程の学費ローンが残ってるの。βテスト報酬の五万ドルがないと学費が払えなくて中退確定。──帰れないと、報酬ももらえない。中退も確定。研究人生も終わり」

「……」

「だから帰りたいんだ。好奇心だけじゃない。金の問題。現実の、泥臭い、くだらない金の問題」

　蓮はLynxの横顔を見た。挑発的で大胆な彼女の、初めて見る素直な表情だった。

「──帰ろう。全員で」

　蓮の声は静かだった。

「全員で帰るぞ」

　Lynxはほんの一瞬、目を逸らした。そして──ニヤリと笑った。

「あんたさ、そういう台詞、似合わないよ」

「分かってる」

「でも──嫌いじゃない」
