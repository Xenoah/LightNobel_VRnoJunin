# 第六話　地下回廊

*Phase 3: 希望の影 ─ EP-006 / VR内 Day 10-15*

---

## 1. Exodus

　Day 11。

　帰還を目指す技術者グループは、正式に名前を得た。

　──Exodus。

　命名したのはLynxだった。

「出エジプト記。囚われの民を約束の地へ導く。……まあ宗教っぽいのがアレだけど、語感はいいでしょ」

「中二病っぽいとも言う」

「うっさいな。あんたに名前つけさせたら『帰還班』とかになるじゃん」

　メンバーは、コアが蓮・Lynx・Milaの三名。それに賛同した技術者が七名。合計十名。ネットワークエンジニア、ソフトウェア開発者、データベース管理者──QAだけでは手が回らない領域をカバーする陣容だった。

　中でも要となるのが、Firewall──本名不明のインフラセキュリティ専門家。二十六歳、カナダ出身。寡黙で無愛想だが、低レイヤーのシステム解析能力は群を抜いている。

「Undercroft最深部まで行く。保守APIの生存を確認する」

　蓮がブリーフィングで告げると、Firewallが静かに手を挙げた。

「API以前に、ネットワーク構成を確認したい。Arcoidaの内部トポロジーが分からなければ、何に繋がっているかも分からない」

「同意だ。探索と並行して、ネットワークスキャンもやる」

## 2. 降下

　The Undercroft。

　Central Nexusの地下に広がる巨大なダンジョンエリア。公式設定では「数千年前の文明の遺跡」とされているが、実態はβテスト用のストレステスト環境──高レベルのモンスター、複雑な地形、罠──が配置されたコンバットゾーンだ。

　Exodusの探索隊──蓮、Lynx、Firewall、技術者二名──が最初の階層に足を踏み入れた。

　暗い。

　天井は高く、壁面には錆びた配管のようなオブジェクトが走っている。足元の石畳は湿っており、所々にコケが光っている──バイオルミネッセンスの演出だ。空気は冷たく、微かに金属の匂いがする。

「レベル設定……エリア平均Lv18。俺たちの平均が12。厳しいな」

「厳しいくらいがちょうどいいんじゃない？」

　Lynxが軽口を叩く。だがゴーグルの向こうの目は真剣だ。

　第一層は比較的安全だった。低レベルのモブを処理しながら進む。蓮とFirewallが前衛、Lynxが後方でネットワークスキャンの装置──NCE経由で再構築した簡易ツール──を操作する。

　第二層で、最初の障害に当たった。

「──接近。複数」

　Firewallの警告と同時に、壁の隙間から三体の四足獣型モンスターが飛び出した。蓮がDay 0で戦った個体と同じ種だが、レベルが二倍以上高い。

「散開！　Lynx、後ろに下がれ！」

　蓮は短剣を構えた。技術者二名が盾持ち──探索用装備として支給されたもの──で前線を張り、蓮が側面から弱点を突く。Firewallは冷静に一体を足止めし、連携で一体ずつ処理していく。

　五分の戦闘で三体を撃破。だが全員のHPが半分以下まで削られた。

「ポーションの在庫、あと七本」

「節約しないとね。……ていうか、ここBerserkersの偵察隊も来てるって噂だよ」

「遭遇したら回避が優先だ。目的はAPIの確認。戦闘は最小限に」

## 3. Advocateの牛歩

　同じ日の午後──Central Nexusでは、全く別の動きが起きていた。

　Advocateが、Loyalistsのメンバーを通じて蓮たちの動きを把握していた。

```
[RSS] #general
@advocate: 皆さんに注意喚起です。
　Undercroftへの無許可探索は危険です。
　高レベルのモンスターが配置されており、気絶のリスクが高い。
　もし探索中に全員が気絶した場合、リスポーンまでの間に
　アイテムやデータを失う可能性があります。
　──外部の救出が進んでいる今、内部で危険を冒す必要はありません。
　待ちましょう。
```

「あー、また始まった」

　Lynxが第三層の通路でRSSを確認しながら舌打ちした。

「Advocate、あたしたちの行動を牽制してるよ。『危険だから行くな』って」

「……善意の可能性もある」

「善意だとしても、結果としてあたしたちの足が遅くなる。LoyalistsのメンバーがExodusの出入りを見張ってるって報告もある」

　蓮は黙った。

　──Advocateの行動は「防衛的」だ。テスターの安全を守りたいだけ──そう解釈できる。だがLynxの言う通り、結果として帰還の試みが遅延している。

　善意か、意図的か。

　蓮にはまだ判断がつかなかった。

## 4. デバッグエリア

　Day 13。

　Exodusは第四層に到達した。ここまでの探索で三日を費やし、メンバーの疲労が蓄積している。ポーションの残りは二本。

　だが──第四層のさらに奥、Lynxが発見した壁の当たり判定のバグを利用して、封鎖された一角に潜り込むと──

「──なんだ、ここは」

　蓮の目が見開いた。

　部屋だった。

　それまでのダンジョンの「遺跡」という世界観から完全に逸脱した空間。白い壁、蛍光灯のように均一な照明、金属製のデスクが並ぶオフィスのような環境。

「デバッグフロアだ」

　Lynxが息を呑んだ。

「β版のリリース前に開発チームが使っていたテスト環境──その残滓よ」

　デスクの上にはターミナルが残っていた。ほとんどは電源が入っていないが、一台だけ──奥の壁際に置かれた端末が、薄暗い画面を点灯させていた。

　蓮がキーボードに触れた。

```
> _
```

　カーソルが点滅している。

「……生きてる」

　蓮はコマンドを入力した。手が震えた。

```
> curl localhost:8080/api/v1/health
```

　三秒。五秒。画面に何も表示されない。十秒──

```
> curl localhost:8080/api/v1/health

{"status":"ok","version":"2.1.0-rc3","modules":["ncb","governance","maintenance"]}
```

　蓮は──三秒間、息ができなかった。

　Lynxが叫んだ。

「生きてる！　保守APIが──まだ動いてる！」

　Firewallが画面を食い入るように見つめた。

「`modules`に`maintenance`がある。……保守モジュールが、まだ稼働している」

　蓮は両手をデスクに突き、深く息を吐いた。

「maintenance……」

　──帰れるかもしれない。

　これは──内部からArcadiaのシステムにアクセスするための入口だ。ガバナンスの構造を調べ、凍結を解除する手段を探し──全員を帰すための、最初の一歩。

「Ren」

　Lynxが蓮の肩を叩いた。

「泣いてんじゃないよ。まだ入口だよ」

「泣いてない」

「嘘つけ。目、赤いじゃん」

　蓮は袖で目元を拭い、画面に向き直った。

## 5. Exodus Base Camp

　Day 15。

　第四層のデバッグフロアを、Exodusは正式に「Base Camp」として設営した。

　ターミナルは四台が復旧。電源系統は独立しており、ガバナンス凍結の影響を受けていないことが確認された。保守APIのドキュメントの断片──APIリファレンスのキャッシュ──も発見された。

　蓮はメンバーを集めた。

「現時点で分かっていること。保守APIのヘルスチェックは生きている。`maintenance`モジュールが稼働中。ドキュメントの断片から、権限昇格のためのトークン生成APIが存在することが示唆されている。だが──アクセスするには認証が必要だ」

「認証レベルは？」

「最低でも`MAINTENANCE_ADMIN`ロール。現在の俺たちのセッショントークンでは権限不足だ」

　Firewallが腕を組んだ。

「トークンの再署名か、暗号解読が必要になる。……時間がかかるな」

「時間はある」蓮は言った。「だが無限ではない。カプセルの物理限界は六十日。……既に十五日が経過している」

　残り四十五日。

　Lynxがターミナルを叩きながら言った。

「ねぇRen。このAPIのレスポンスに`governance`モジュールがあったでしょ。……ガバナンスコントラクトに内部からアクセスできるなら、凍結の構造が分かるかも」

「ああ。そこが次のステップだ」

　蓮は壁に貼られた手書きの地図を見た。Undercroftの構造、各階層のモンスター配置、Base Campの位置──すべてが手作業で作られた、粗くて不完全な地図。だが確かな第一歩。

## 6. 帰り道

　Day 15の夜。Base CampからCentral Nexusへ戻る道中。

　最後尾を歩いていた蓮に、Lynxが追いついた。

「ねぇRen」

「ん？」

「あたしさ、正直に言うと──帰る理由、半分は不純なんだよね」

「……どういう意味だ」

「博士課程の学費ローン。βテストの報酬──五万ドル。帰還しないと支払われない契約になってる。帰れなかったら中退確定」

　蓮は黙って歩き続けた。

「不純でしょ。命がかかってるのにお金の話してるの」

「……不純じゃない」

「え？」

「帰る理由は人それぞれだ。崇高な動機だけが正しいわけじゃない。五万ドルのために帰りたい──それでいい。動機なんか何でもいい。帰る、ということさえ変わらなければ」

　Lynxはしばらく黙り、それから笑った。

「あんたさ、不器用だけどいいこと言うよね」

「うるさい」

「もっと褒めて」

「帰れ」

「帰ってるよ。……帰るんだよ。──全員で」

　二人は黙って歩き続けた。第三層の長い通路、苔の光がぼんやりと壁を照らしている。

　蓮は──ふと、足を止めた。

　Lynxも止まった。

「どうした？」

「……いま、足音が聞こえなかったか」

「足音？　あたしたちの足音でしょ」

「俺たちは五人だ。だが──さっき、六人分の足音が聞こえた」

　Lynxが振り返った。通路の闇は深く、何も見えない。

「……Ren。疲れてるんだよ。三日間Undercroftにいたんだから、聴覚の閾値が下がってるの」

「……そうだな」

　蓮は歩き出した。全員が歩き出した。

　足音が──五つ。

　蓮は耳を澄ました。五つ。──五つだ。

　だが一瞬前まで──確かに──六つだった。

　止まると、消える。
