# 第十話　同時多発作戦

*Phase 5: 転落 ─ EP-010 / VR内 Day 28-29*

---

## 1. 配置

　Day 28、深夜。

　各チームの最終ブリーフィングがBase Campで行われた。

「全二十一ノード、七チーム。各チーム三ノードを担当」

　蓮はホワイトボードに描いた作戦図を指差した。

「Alpha──RenとLynxのチーム。Central NexusのNode-01, 02, 03を担当。Bravo──FirewallのチームがUndercroftのNode-04, 05, 06。Charlie──Bastionが率いるSettlers部隊がVerdant ExpanseのNode-07, 08, 09──」

　七チーム、全四十六人。The Awakened から帰還を選んだ者も含まれている。Fenrisの元Berserkersが各チームに戦闘護衛として分散配置された。

「作戦開始は明日──Day 29の正午。全チームが同時にノードへアクセスリクエストを送る。十四台以上の確認が取れた時点で`MAINTENANCE_OVERRIDE_HALT`を実行する」

「もし十四台に届かなかったら？」Bastionが問う。

「……届かせる」

## 2. 正午

　Day 29、12:00。

　Arcadia全域で、同時に動きが始まった。

　蓮はCentral Nexusの地下端末室に走り込んだ。Lynxが隣でキーボードを叩き、蓮が確認コマンドを送信する。

```
> api/v1/validators/node-01/confirm
Response: {"status": "CONFIRMED", "timestamp": "..."}

> api/v1/validators/node-02/confirm
Response: {"status": "CONFIRMED", "timestamp": "..."}

> api/v1/validators/node-03/confirm
Response: {"status": "CONFIRMED", "timestamp": "..."}
```

「Alpha完了！　三台確認！」

　RSSにBravo──Firewallのチームからも報告が入る。

```
[RSS] #operation
@firewall: Bravo完了。Node-04, 05, 06 CONFIRMED。
@bastion_settlers: Charlie完了。Node-07, 08, 09 CONFIRMED。
```

「九台。順調だ──」

　だがDeltaチームから不穏なメッセージが入った。

```
[RSS] #operation
@delta_lead: Delta問題発生。Ethereal Gardensの入口でThe Awakened過激派が妨害。
　物理的に通路を封鎖されてます。
```

## 3. 妨害

　The Awakenedの過激派──帰還を「退化」と信じる少数の狂信者──が、EthReal Gardensの主要通路にバリケードを築いていた。五人。武器を持っている。

「帰還は認めない！　この世界を壊すな！」

　DeltaチームのリーダーがRSSで状況を報告する。蓮はBase Campからリアルタイムで指示を出した。

「Fenris、予備戦力を投入できるか」

「任せろ」

　Fenrisが率いる三人の元Berserkersが、Deltaチームに合流した。傭兵の実力は圧倒的で、十五分で過激派を制圧。Deltaチームはノードへのアクセスに成功した。

```
[RSS] #operation
@delta_lead: Delta完了。Node-10, 11, 12 CONFIRMED。妨害排除済み。
```

「十二台」

　残り三チーム分──九ノード。うち二台が確認されれば十四に届く。

```
@echo_lead: Echo完了。Node-13 CONFIRMED。
@foxtrot_lead: Foxtrot、Node-16 CONFIRMED。Node-15はアクセス遅延。再試行中。
```

「十四台──あと一つで──」

　そしてGolfチーム──Advocateが担当するNode-17, 18, 19──からの報告が入った。

```
@advocate: Golf進行中。Node-18 CONFIRMED。Node-19 CONFIRMED。
　Node-17は──応答が遅い。少し待ってほしい。
```

　蓮は時計を見た。残り時間は──十分。ノードの確認リクエストにはタイムアウトがある。全ノードの確認が三十分以内に完了しなければ、最初のノードから再確認が必要になる。

```
@advocate: Node-17、応答なし。引き続きリトライ中。
```

「Foxtrotも──Node-15がまだだ」

　Foxtrotチームから追加報告──

```
@foxtrot_lead: Node-15 CONFIRMED！
```

「これで十四──いや──」

　蓮が計算し直す。確認されたのは──Node-01〜13, 15, 16, 18, 19。合計十七台。十四台を超えている。──だが──

「待って」

　Firewallが声を上げた。

「Node-01〜12は確認済み。Node-13, 15, 16は確認済み。Node-18, 19は確認済み。──でもNode-14, 17, 20, 21がまだだ。十七台中──いや違う。Echoチームの報告を再確認──」

　混乱が生じていた。同時多発的な通信が交錯し、どのノードが確認済みでどのノードが未確認かの集計に齟齬が出ている。

　Firewallが冷静にログを整理した。

```
確認済み: 01, 02, 03, 04, 05, 06, 07, 08, 09, 10, 11, 12, 13 → 13台
　　　　  15, 16, 18, 19 → +4 = 合計17台

未確認:   14, 17, 20, 21 → 4台
```

「十七台確認。十四は超えてる──」

「待って」Firewallの声が鋭くなった。「Node-17の応答を確認する。……エラーが返ってきてる」

```
> api/v1/validators/node-17/status

{"node": "node-17", "status": "CONFIRMATION_REJECTED", 
 "reason": "token_expiry_check_updated", 
 "message": "This node has been patched against known token exploits."}
```

　蓮の血が引いた。

「Node-17が──トークンの期限チェックを修正済み？」

「俺たちが使ったトークンバイパスの手法が──このノードだけブロックされてる」

　Firewallが画面を凝視した。

「誰かが──このノードだけ──パッチを当てた」

## 4. Firewall

　Firewallの指が止まった。

「Ren。……このパッチの適用タイムスタンプ──三十分前だ」

「三十分前──作戦開始直後？」

「作戦開始後に、誰かがNode-17のトークン検証ロジックを修正している。意図的に──俺たちの手法を無効化するために」

　蓮は周囲を見回した。Base Campにいるメンバーの顔を──

「──内部犯か」

「そうとしか考えられない。作戦の技術詳細を知っていて、かつNode-17に直接アクセスできる権限を持つ者──」

　Firewallは画面に食い入りながら、低い声で続けた。

「十三台の確認で閾値十四に一つ足りない──もし十七台のうちNode-17だけが拒否されれば──」

「いや、十七台確認で十四は超えてる。Node-17なしでも──」

「違う」Firewallが首を振った。「確認済みの十七台のうち、Node-14, 20, 21の三台は──まだ最終確認を受けていない。Echoチームの報告が混線していた。正確には──」

　Firewallが再集計した。

```
最終確認: 13台 (Node-01〜13)
暫定確認: Node-15, 16, 18, 19 → 処理完了確認待ち
拒否:     Node-17
```

「プロセスの完了を待つと──最大で十七台。だがNode-17が拒否のままなら、残りのノードの確認プロセス完了後に再検証が入る。タイムアウトまで──」

「あと何分だ」

「八分」

　Firewallが──もう一つ、画面を指差した。声が低くなった。

「Ren。……このパッチの適用署名──見てくれ」

```
patch_applied_by: USER-000
signature_type: internal_maintenance
timestamp: 2031-07-29T12:02:11Z
```

「`USER-000`。──テスターのIDはCHAR-001からCHAR-048。開発者IDはDEV-001から。保守IDはSYS-001から。……`USER-000`はどのカテゴリにも存在しないIDだ」

「Advocateのセッションから適用したのでは？」

「Advocateの内部IDはCHAR-012。こっちはUSER-000──別の存在だ」

　蓮は画面を凝視した。

　──誰が、このパッチを当てた。

　テスターでもなく、開発者でもなく、保守システムでもない──「USER-000」。ゼロ。存在の起点。

　──あるいは、存在しないはずのもの。
