```
バリデーターノード配置図：

Central Nexus    : Node-01, Node-02, Node-03
Verdant Expanse  : Node-04, Node-05, Node-06
Crimson Peaks    : Node-07, Node-08, Node-09

The Undercroft   : Node-14, Node-15
Ether Reach      : Node-16, Node-17
Skyward Summit   : Node-18, Node-19

「七つのリージョンに分散配置されている。チームを分けて、同時並行で確認を取る。四時間以内に十四ノードの確認を完了させる」

　Lynxが補足した。

「タイムリミットは四時間。理由は二つ。一つ目──`MAINTENANCE_ADMIN` トークンの有効期限が四時間。二つ目──四時間を超えると、Lethe Chainの異常検知が発火し、ノードが自動リセットされる可能性がある」

　蓮はチーム編成を発表した。

```
Alpha : Ren, Lynx          → Central Nexus (Node-01〜03)

「全チームにRSS中継器を配布する。進捗はリアルタイムで共有。──各自、持ち場へ」

## 2. 発進

　Day 29、午前六時。作戦開始。

　Central Nexusのハブから、テスターたちが散っていった。各リージョンへの転送ゲートが光り、一人ずつ消えていく。

　蓮はLynxと共にNode-01に向かった。Central Nexusの地下──行政棟の最深部。通常はNPCの管理者しか入れないエリアだが、`MAINTENANCE_ADMIN` ロールで扉が開いた。

　Node-01のコンソールは、黒い立方体だった。高さ一メートル。表面に、星のように光るLEDが散在している。各LEDがトランザクションの処理状況を示している。

「Node-01。コマンドを入力する」

```
> maintenance_override_halt --phase=validate --node=01
Validating... OK
Node-01 confirmation: GRANTED
Remaining: 13/14
```

「──一つ目、確認」

　RSSに報告が飛ぶ。

```
[Alpha / 06:12] Node-01 GRANTED. 残り13
[Beta  / 06:18] Node-04 GRANTED. 残り12
[Gamma / 06:25] Node-10 GRANTED. 残り11
[Delta / 06:31] Node-14 GRANTED. 残り10
```

## 5. 時間切れの恐怖

　午前九時。トークンの有効期限まで──残り一時間。

　十三ノード確認済み。あと一つ。

　だが最後の一つが──突破できない。

「Lynx。Node-17以外のノードで代替できるか？」

「調べた。Node-21とNode-22に未確認のセカンダリノードがある。──でもそこに到達するにはSkyward Summitの頂上を越える必要がある。一時間じゃ──」

「足りない」

　蓮は壁に拳を叩きつけた。

　──十三ノード。あと一つ。たった一つのノードが、四十八人の帰還を阻んでいる。

「エミリア──Milaに聞ける方法はないか」

「……あります」

　エミリアの声が震えていた。

「プロンプトインジェクション脆弱性を使えば、NCEの内部からNode-17のファームウェア設定を読み出せるかもしれない。Node-17の認証ロジックのバイパス方法を──」

「やってくれ」

「──でもRen。脆弱性を深く突けば突くほど、Milaの──セッション整合性が──」

「分かっている」

　蓮の声は冷たかった。自分でも分かっている。道具ではないものを、道具として使おうとしている。

「──やってくれ。頼む」

　エミリアは端末に向かった。
