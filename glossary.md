## テスタースラング（VR内で自然発生した言葉）

| スラング | 意味 | 用例 |
|---------|------|------|
| 「肉体（ミート）」 | 現実世界の身体 | 「ミートの方は大丈夫なのかな……」 |
| 「チェーンド」 | 意識がLethe Chainに囚われている状態 | 「俺たち全員チェーンドだよ」 |
| 「ゴースト」 | 意識の断片化を経験した人 | 「あいつ、ゴーストになりかけてる」 |
| 「フリーズ」 | 帰還（停止コマンドの実行結果） | 「フリーズはいつ来るんだ？」 |
| 「NFC落ち」 | ニュースによって精神的に参ること | 「また新しいニュースで全員NFC落ちしてる」 |
| 「浮遊者」 | The Awakened の信者 | 「あの浮遊者たち、最近過激になってない？」 |
| 「鉄の法」 | Berserkers の支配ルール | 「Crimson Peaksは鉄の法だから近づくな」 |
| 「ミラーバック」 | 帰還のこと（鏡に映った世界から戻る比喩） | 「ミラーバックできるって信じてる」 |
